# 1. 주문서 조회

## 6-1. 주문서 조회

**\[주문 관리] > \[주문서 등록/조회]** 메뉴에서 등록된 주문서 목록을 확인할 수 있다.

***

{% hint style="info" %}
손가락으로 화면을 벌려 메뉴얼의 설명 이미지를 확대할 수 있다.
{% endhint %}



<figure><img src="../.gitbook/assets/6-1. 주문서 조회(1).png" alt=""><figcaption></figcaption></figure>



![](../.gitbook/assets/1.png) **\[주문 관리] > \[주문서 등록/조회] 메뉴에서 주문서 목록을 확인할 수 있다.**

![](../.gitbook/assets/2.png) **주문서를 클릭하면 상세 정보를 확인할 수 있다.**

![](../.gitbook/assets/3.png) **고객 이름, 고객 전화번호, 주문번호로 주문서를 검색할 수 있다.**

![](../.gitbook/assets/4.png) **화살표를 클릭하여 진행 상태별로 주문서를 정렬할 수 있다.**

{% hint style="info" %}
**주문서 상태**

* 유효: 고객이 정상적으로 주문을 완료하여 시스템에 등록된 상태로, 수정 및 취소가 가능함
* 완료: 주문부터 설치까지 모든 프로세스가 성공적으로 완료된 상태로, 수정 및 취소가 불가능함
* 취소: 주문 프로세스가 중단된 상태로, 추가 처리나 수정이 불가능함
{% endhint %}
