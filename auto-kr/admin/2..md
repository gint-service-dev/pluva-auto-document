# 2. 주문 취소

## 6-2. 주문 취소

발급 상태의 주문서만 취소할 수 있으며, 취소된 주문은 되돌릴 수 없다. 취소 시 해당 주문과 연결된 설치티켓도 함께 취소된다.

***

{% hint style="info" %}
손가락으로 화면을 벌려 메뉴얼의 설명 이미지를 확대할 수 있다.
{% endhint %}



{% stepper %}
{% step %}
#### \[주문 관리] > \[주문서 등록/조회] 메뉴에서 주문서를 클릭하여 상세 화면에 접속한다.

<figure><img src="../.gitbook/assets/6-2. 주문서 취소(1).png" alt=""><figcaption></figcaption></figure>

{% hint style="danger" %}
유효 상태의 주문서만 취소할 수 있다. 취소한 주문은 되돌릴 수 없으므로 충분히 확인 후 진행해야 한다.
{% endhint %}

{% hint style="danger" %}
설치 중인 티켓이 있다면 주문을 취소할 수 없다.
{% endhint %}
{% endstep %}

{% step %}
#### \[전체 주문 취소] 버튼을 클릭한다.

<figure><img src="../.gitbook/assets/6-2. 주문서 취소(2).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### 주문서가 취소 상태로 변경된다. 연관된 설치티켓도 모두 취소된다.

<figure><img src="../.gitbook/assets/6-2. 주문서 취소(3).png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
