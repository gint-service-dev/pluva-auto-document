# 3. 설치티켓 조회

## 6-3. 설치티켓 조회

설치티켓 조회 기능은 등록된 설치 일정과 각각의 설치 진행 상태를 확인하고, 티켓별 상세 정보를 열람할 수 있는 화면이다.

***

{% hint style="info" %}
손가락으로 화면을 벌려 메뉴얼의 설명 이미지를 확대할 수 있다.
{% endhint %}



<figure><img src="../.gitbook/assets/6-3. 설치 티켓 조회(1).png" alt=""><figcaption></figcaption></figure>



![](../.gitbook/assets/1.png) **\[주문 관리] > \[설치티켓 조회] 메뉴에서 티켓 목록을 확인할 수 있다.**

![](../.gitbook/assets/2.png) **티켓을 클릭하면 상세 정보를 확인할 수 있다.**

![](../.gitbook/assets/3.png) **고객 이름, 고객 전화번호, 담당자 이름, 설치티켓번호로 티켓을 검색할 수 있다.**

![](../.gitbook/assets/4.png) **화살표를 클릭하여 진행 상태별로 티켓을 정렬할 수 있다.**

{% hint style="info" %}
설치티켓 상태

* 발급: 설치 작업을 위해 티켓이 생성된 상태로, 취소 처리가 가능함
* 설치중: 현장에서 easy setup으로 설치를 진행 중인 상태
* 완료: easy setup으로 설치가 완료된 상태
* 취소: 설치 작업이 중단되거나 취소된 상태로, 해당 티켓으로는 더 이상 설치 진행이 불가능함
{% endhint %}
