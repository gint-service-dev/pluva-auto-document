# 4. 설치티켓 취소

## 6-4. 설치 티켓 취소

설치 진행 중인 상태의 티켓은 관리자(Admin)가 직접 취소할 수 있다. 단, 티켓 취소는 되돌릴 수 없으므로 반드시 상태를 확인한 후 진행해야 한다.

***

{% hint style="info" %}
손가락으로 화면을 벌려 메뉴얼의 설명 이미지를 확대할 수 있다.
{% endhint %}



{% stepper %}
{% step %}
#### \[주문 관리] > \[설치티켓 조회] 메뉴에서 설치티켓을 클릭하여 상세 화면에 접속한다.

<figure><img src="../.gitbook/assets/6-4. 설치티켓 취소(1).png" alt=""><figcaption></figcaption></figure>

{% hint style="danger" %}
유효 상태의 주문서만 취소할 수 있다. 취소한 주문은 되돌릴 수 없으므로 충분히 확인 후 진행해야 한다.
{% endhint %}

{% hint style="danger" %}
설치 중인 티켓이 있다면 주문을 취소할 수 없다.
{% endhint %}
{% endstep %}

{% step %}
#### \[티켓 취소] 버튼을 클릭한다.

<figure><img src="../.gitbook/assets/6-4. 설치티켓 취소(2).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### 티켓이 취소 상태로 변경된다.

<figure><img src="../.gitbook/assets/6-4. 설치티켓 취소(3).png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
