# 5. 고객 계정 조회

## 6-5. 고객 계정 조회

고객 계정 조회 기능은 플루바 오토 관리자(Admin)가 등록된 고객 계정의 정보를 확인하고, 각 계정의 사용 상태 및 연결된 기기 정보를 관리할 수 있는 화면이다.

***

{% hint style="info" %}
손가락으로 화면을 벌려 메뉴얼의 설명 이미지를 확대할 수 있다.
{% endhint %}



<figure><img src="../.gitbook/assets/6-5. 고객 계정 조회(1).png" alt=""><figcaption></figcaption></figure>



![](../.gitbook/assets/1.png) **\[고객 관리] > \[계정 조회] 메뉴에서 플루바오토 계정 목록을 확인할 수 있다.**

![](../.gitbook/assets/2.png) **계정을 클릭하면 상세 정보를 확인할 수 있다.**

![](../.gitbook/assets/3.png) **이름, 전화번호로 계정을 검색할 수 있다.**

{% hint style="info" %}
운영 계정은 사용 용도에 따라 총 3가지 타입으로 분류되며, 원하는 타입을 선택해 볼 수 있다.

* ![](../.gitbook/assets/데모.png) : 플루바 오토를 무상으로 일정 기간 고객에게 제공하거나, 판촉 활동을 위해 본사/대리점에서 일시적으로 사용하는 계정
* ![](../.gitbook/assets/렌탈.png) : 플루바오토를 유상으로 일정 기간 고객에게 임대할 때 고객이 사용하는 계정
* ![](../.gitbook/assets/테스트.png) : 테스트 목적으로 사용하는 계정
{% endhint %}

{% endhint %}
