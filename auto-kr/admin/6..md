# 6. 대시보드

## 6-6. 대시보드

대시보드는 **플루바 오토 장비의 위치 및 상태를 실시간으로 확인**할 수 있는 화면입니다.\
지도 기반으로 각 장비의 연결 상태(Online/Offline), 위치 좌표, 사용 이력 등을 시각적으로 표시하여 관리자 또는 기술 담당자가 신속하게 장비의 상태를 파악할 수 있다.

***

{% hint style="info" %}
손가락으로 화면을 벌려 메뉴얼의 설명 이미지를 확대할 수 있다.
{% endhint %}



<figure><img src="../.gitbook/assets/6-6. 대시보드(1).png" alt=""><figcaption></figcaption></figure>



![](../.gitbook/assets/1.png) **상단 요약 정보 : 관리 중인 모든 메인컨트롤러 수와 오늘 사용된 메인컨트롤러 수를 표시한다.**

![](../.gitbook/assets/2.png) **지도 : 관리 중인 메인컨트롤러의 위치를 표시한다.**

> * **초록색 마커: 현재 사용 중인 위치**
> * **빨간색 마커: 마지막으로 사용한 위치**

![](../.gitbook/assets/3.png) **고객 목록 : 고객이 사용 중인 메인컨트롤러의 정보를 간략히 표시한다.**

![](../.gitbook/assets/4.png) **상태**

> * ONLINE: 메인컨트롤러의 전원이 켜져 있고 네트워크가 연결된 상태
> * OFFLINE: 메인컨트롤러의 전원이 꺼져 있거나 네트워크 연결이 끊어진 상태

![](../.gitbook/assets/5.png) **Live (파란색) : 작업 진행 상황을 실시간으로 확인할 수 있는 기능**

> Status가 "ONLINE"인 단말기에서만 가능

![](../.gitbook/assets/6.png) **History : 과거 사용 이력을 조회할 수 있는 기능**

> “ONLINE”, “OFFLINE” 모두 조회 가능
