# 7. 실시간 모니터링

## 6-7. 실시간 모니터링

실시간 모니터링 기능은 메인컨트롤러가 **연결된 상태(ONLINE)** 일 때 장비의 위치, 움직임 상태, 작업 진행 여부 등을 실시간으로 확인할 수 있는 화면이다.

***

{% hint style="info" %}
손가락으로 화면을 벌려 메뉴얼의 설명 이미지를 확대할 수 있다.
{% endhint %}

#### 대시보드의 고객 목록에서 <mark style="color:$primary;">**파란색 \[Live]**</mark> 버튼을 클릭하여 실시간 사용 현황을 볼 수 있다. 메인컨트롤러가 켜져 있을 때만(ONLINE 상태에서만) 확인할 수 있다.

***

![](../.gitbook/assets/6.png) **메인컨트롤러 상태**

> * GPS Fix (파랑): GPS가 정상적으로 연결된 상태
> * Straight Point A (보라): 주행 모드에 진입하여 AB라인을 그리는 상태
> * AB Completed (노랑): 자율주행이 가능한 상태
> * Straight Auto (초록): 자율주행 중
> * Straight Restrict (빨강): 자율주행이 제한된 상태

![](../.gitbook/assets/7.png) **메인컨트롤러 사용 위치**

![](../.gitbook/assets/8.png) 주행 속도, 주행 방향

![](../.gitbook/assets/9.png) 거리 오차, 방향 오차

![](../.gitbook/assets/10.png) 속도 그래프

> * 가로축 - 시간, 세로축 - 주행 속도
> * 실시간 모니터링 중 메인컨트롤러의 전원이 꺼지면 더이상 그래프 선이 그려지지 않는다.

![](../.gitbook/assets/11.png) 지도

> * 주행 경로 및 상태 표시
>   * 파랑: GPS가 정상적으로 연결된 상태
>   * 보라: 주행 모드에 진입하여 AB라인을 그리는 상태
>   * 노랑: AB라인 생성 완료. 자율주행이 가능한 상태
>   * 초록: 자율주행 중
>   * 빨강: 자율주행이 제한된 상태
> * 주행 속도, 방향 오차, 거리 오차 표시
> * 지도 확대/축소, 전체화면 보기 가능



<figure><img src="../.gitbook/assets/6-7. 실시간 모니터링(3).png" alt=""><figcaption></figcaption></figure>

![](../.gitbook/assets/12.png) **차량 제어 설정 값**

> * 차량 제어 설정값은 펌웨어 버전이 25.09.29 버전부터 표시된다.
> * 데이터 표기 방식
>   * \[기능 번호] - 이름 - 값 표시
>   * 커브 제어는 AB 커브 모드에서만 지원
> * 데이터 보기만 가능. 수정은 불가
