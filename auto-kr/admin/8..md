# 8. 지난 작동기록 조회

## 6-8. 지난 작동기록 조회

실시간 모니터링 기능은 메인컨트롤러가 **연결된 상태(ONLINE)** 일 때 장비의 위치, 움직임 상태, 작업 진행 여부 등을 실시간으로 확인할 수 있는 화면입니다.

***

{% hint style="info" %}
손가락으로 화면을 벌려 메뉴얼의 설명 이미지를 확대할 수 있다.
{% endhint %}

{% stepper %}
{% step %}
#### 대시보드의 고객 목록에서 \[History] 버튼을 클릭한다.

<figure><img src="../.gitbook/assets/6-8. 지난 작동 기록 조회(1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### 원하는 일시를 찾아 \[View details]를 클릭한다.

<figure><img src="../.gitbook/assets/6-8. 지난 작동 기록 조회(2).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### SELECT AREA에서 원하는 구간을 드래그하여 선택 후 재생 버튼을 눌러 작동 기록을 확인한다.

<figure><img src="../.gitbook/assets/6-8. 지난 작동 기록 조회(3).png" alt=""><figcaption></figcaption></figure>

![](../.gitbook/assets/1.png) **메인컨트롤러 제품키, 작동 시간**

![](../.gitbook/assets/2.png) **속도 그래프**

> * 가로축 - 시간, 세로축 - 주행 속도
> * \[CLEAR SELECTION]을 누르면 선택한 구간이 초기화된다.

![](../.gitbook/assets/3.png) **재생 컨트롤러**

> * 재생 속도를 1배, 2배, 4배, 8배로 조절할 수 있다.

![](../.gitbook/assets/4.png) **메인컨트롤러 상태**

> * GPS Fix (파랑): GPS가 정상적으로 연결된 상태
> * Straight Point A (보라): 주행 모드에 진입하여 AB라인을 그리는 상태
> * AB Completed (노랑): 자율주행이 가능한 상태
> * Straight Auto (초록): 자율주행 중
> * Straight Restrict (빨강): 자율주행이 제한된 상태

![](../.gitbook/assets/5.png) **주행 속도, 거리 오차, 방향 오차, 연결된 위성 수**

![](../.gitbook/assets/6.png) **주행 경로 및 상태 표시 지도 확대/축소, 전체화면 보기 가능**
{% endstep %}
{% endstepper %}
