# 0. 고객 안내사항

## 5-0. 고객 안내사항

설치 후 고객에게 서비스 이용 조건을 정확히 전달함으로써 지속적으로 원활한 서비스를 이용할 수 있도록 한다.

***

### a. LTE 유심 서비스 및 데이터 이용 가이드&#x20;

* **메인컨트롤러 구매 시 6개월 무료 LTE 유심(10GB)이 서비스로 제공된다.** \
  만료 후에는 신규 유심으로 교체하도록 안내한다.
* **태블릿에는 유심이 제공되지 않으므로** 별도의 유심이나 휴대폰 테더링(핫스팟)을 사용하도록 안내한다.
* 플루바오토 서비스는 **일반적으로 월 1GB, 집중적으로 매일 사용 시 월 2GB의 데이터를 소모**한다. 고객의 사용 패턴을 고려하여 적절한 요금제를 안내한다.



### b. 무선 소프트웨어 업데이트 (OTA software update, OTA: Over-The-Air)

{% hint style="info" %}
OTA란?

무선 네트워크를 통해 자율주행에 필요한 소프트웨어를 **원격으로 업데이트하는 기능이다.** \
**별도의 방문이나 수동 설치 없이** 플루바오토를 최신 상태로 유지할 수 있다.
{% endhint %}

OTA 업데이트는 다음 상황에서 진행된다. 업데이트 일정은 사전에 안내된다.

1. 신규 기능이 추가되었을 때
2. 중요한 문제점이 발견되었을 때
3. 신규 기능의 정식 출시 전 베타 테스트를 진행할 때: 베타 테스트 희망 고객에게만 사전 공지 후 배포



### c. 최초 실행 시 OTA 진행OTA란?

{% hint style="info" %}
* **플루바오토를 처음 사용할 때 자동으로 업데이트가 진행된다.** \
  최초 업데이트 중에는 전원을 끄거나 원터치 스위치 또는 플루바오토 앱을 조작하지 않는다.
{% endhint %}

1. 스위치를 눌러 플루바오토의 전원을 켠다.
2. 전원이 켜지면 업데이트가 자동으로 진행된다. 업데이트 진행 상황은 원터치 스위치 화면에서 확인할 수 있다.

> 메인컨트롤러에 유심을 처음 장착한 후 전원을 켜면 네트워크 연결에 시간이 소요될 수 있다.\
> 일정 시간이 지나도 LTE가 연결되지 않는 경우 메인컨트롤러의 전원을 껐다가 다시 켠다.

3. 업데이트는 일반적으로 약 5-10분이 소요되며, 통신 상태가 원활하지 않은 경우 최대 30분까지 소요될 수 있다.
4. 업데이트가 완료되면 플루바오토 앱의 \[장비] > \[메인컨트롤러]에서 변경된 소프트웨어 버전을 확인할 수 있다.
