### b. 골띄기 유턴 설정

#### ➊ 설정 항목 설명

{% columns %}
{% column width="50%" %}


<figure><img src="../.gitbook/assets/5-11. 유턴(2)-(1).png" alt="" width="279"><figcaption></figcaption></figure>


{% endcolumn %}

{% column width="50%" %}


![](../.gitbook/assets/1.png) **자동 유턴을 ON/OFF로 끄고 킬 수 있다.**

> **On:** 작업 영역을 벗어나지 않는 위치에서 자동으로 유턴하거나 ![](<../.gitbook/assets/유턴 아이콘.png>) 버튼을 누른 시점에 유턴한다.\
> **Off:** ![](<../.gitbook/assets/유턴 아이콘.png>) 버튼을 누른 시점에 유턴한다.

![](../.gitbook/assets/2.png) **작업 경계선으로부터 얼마나 떨어진 지점에서 유턴할지 설정한다.** 0.1m 단위로 설정할 수 있다.

<figure><img src="../.gitbook/assets/5-11. 유턴(2)-(2).png" alt="" width="245"><figcaption></figcaption></figure>

![](../.gitbook/assets/3.png) **유턴할 때의 최소 회전 반경을 설정한다.** 0.1m 단위로 설정할 수 있다.

![](../.gitbook/assets/4.png) **직진 주행 후 첫 번째로 회전할 방향을 설정한다.**

![](../.gitbook/assets/5.png) **유턴할 때의 최대 속도를 설정한다**. 안전한 작업을 위해 5km/h 이하의 속도를 권장한다.

{% hint style="info" %}
![](<../.gitbook/assets/Frame 34312.png>) 아이콘을 누르면 해당 기능에 대한 설명을 볼 수 있다.
{% endhint %}
{% endcolumn %}
{% endcolumns %}

> **3) 자율주행 작업 중** ![](../.gitbook/assets/새로고침.png) **버튼을 누르면, 자율 작업 중인 라인 또는**\
> **버튼을 누른 후 첫 자율 작업 라인부터 설정한 회전 방향으로 유턴한다.**
