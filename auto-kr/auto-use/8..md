1. **전진 경로 진입 거리**\
   : 전진으로 차량이 경로 상 얼마나 먼 지점을 목표로 진입할지 설정한다.



2. **후진 경로 진입 거리**\
   : 후진으로 차량이 경로 상 얼마나 먼 지점을 목표로 진입할지 설정한다.

{% hint style="info" %}
경로 진입 거리는 고급 설정에서 수치로 상세히 설정할 수 있으며, 일반 설정과 변경 값이 연동된다.
{% endhint %}



3. **경로 진입 각도**

: 차량이 경로에 진입할 때 얼마나 적극적으로 들어갈지 설정한다. <mark style="color:$primary;">**(증감 권장 단위: 5)**</mark>

{% hint style="info" %}
값이 클수록 경로에 급격하게 진입하고 값이 작을수록 완만하게 진입한다.
{% endhint %}

{% hint style="danger" %}
너무 작게 설정하면 진입이 지나치게 늦어질 수 있다.
{% endhint %}



4.  **경로 진입 시간**

    : 경로 진입을 위해 미리 계획할 시간을 설정한다. <mark style="color:$primary;">**(증감 권장 단위: 0.5)**</mark>

{% hint style="info" %}
값이 클수록 시간을 두고 계획해서 부드럽게 진입하고 값이 작을수록 직전에 계획해서 빠르게 진입한다.
{% endhint %}

{% hint style="danger" %}
진입 거리를 먼저 수정한 후 각도와 시간을 조정해야 한다.
{% endhint %}
{% endcolumn %}

2.  **후진 경로 추종 민감도**

    : 후진으로 진입 후 경로를 얼마나 정밀하게 따라갈지 설정한다. <mark style="color:$primary;">**(증감 권장 단위: 0.2)**</mark>

{% hint style="info" %}
값이 클수록 여유롭게 추종하고 값이 작을수록 민첩하게 추종한다.
{% endhint %}

{% hint style="danger" %}
이 설정은 주행 성능에 큰 영향을 미치므로 신중하게 조정한다.

2.  **랜딩 전 후진 조향 민감도**

    : 후진으로 경로에 완전히 진입하기 전 조향 민감도를 설정한다. <mark style="color:$primary;">**(증감 권장 단위: 0.5)**</mark>



3.  **랜딩 후 전진 조향 민감도**

    : 전진으로 경로에 완전히 진입한 후 조향 민감도를 설정한다. <mark style="color:$primary;">**(증감 권장 단위: 0.5)**</mark>



4.  **랜딩 후 후진 조향 민감도**

    : 후진으로 경로에 완전히 진입한 후 조향 민감도를 설정한다. <mark style="color:$primary;">**(증감 권장 단위: 0.5)**</mark>

{% hint style="info" %}
값이 클수록 핸들 조작이 부드러워지고 값이 작을수록 핸들 조작이 민감해진다.

5. **데드존 보정**

: 핸들을 돌려도 바퀴가 움직이지 않는 구간을 보정한다. <mark style="color:$primary;">**(증감 권장 단위: 5)**</mark>

{% hint style="info" %}
설정값과 실제 데드존의 오차가 클수록, 데드존의 범위가 클수록 투박하게 제어될 수 있다.
{% endhint %}

{% hint style="danger" %}
부정확한 값은 자율주행 성능에 큰 영향을 미치므로 데드존은 반드시 실측하여 정확하게 설정한다.
{% endhint %}



6. **유턴 조향 보정**

: 유턴 시 설정된 경로를 기준으로 조향을 보정한다. <mark style="color:$primary;">**(증감 권장 단위: 0.5)**</mark>

{% hint style="info" %}
값이 클수록 여유롭게 추종하고 값이 작을수록 민첩하게 추종한다.
{% endhint %}

{% hint style="danger" %}
이 설정은 주행 성능에 큰 영향을 미치므로 신중하게 조정한다.

1. **전진 시작 조건 사용**

: 전진으로 자율주행을 시작하기 위한 안전 조건 사용 여부를 설정한다.

{% hint style="info" %}
사용함: 안전 조건을 만족해야만 자율주행을 시작 /사용 안함: 언제든지 자율주행을 시작
{% endhint %}

{% hint style="danger" %}
안전 조건을 사용하지 않으면 위험한 상황에서도 자율주행이 시작될 수 있다. 특별한 경우가 아니라면 안전 조건을 켜둔 상태로 사용한다.
{% endhint %}



2. **전진 거리 조건**

: 전진으로 자율주행을 시작할 수 있는 최대 거리를 설정한다.

{% hint style="info" %}
차량이 경로로부터 이 값보다 먼 거리에 위치하면 자율주행을 시작할 수 없다.
{% endhint %}



3. **전진 각도 조건**

: 전진으로 자율주행을 시작할 수 있는 최대 각도 차이를 설정한다. <mark style="color:$primary;">**(증감 권장 단위: 5)**</mark>

{% hint style="info" %}
차량과 경로의 각도 차이가 이 값보다 크면 자율주행을 시작할 수 없다.
{% endhint %}



4. **전진 속도 조건**

: 전진으로 자율주행을 시작할 수 있는 최대 속도를 설정한다. <mark style="color:$primary;">**(증감 권장 단위: 1)**</mark>

{% hint style="info" %}
이 속도보다 빠르게 주행 중일 때는 자율주행을 시작할 수 없다.
{% endhint %}
