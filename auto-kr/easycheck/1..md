# 1. 계정/제품 정보 확인

## 7-1. 계정/ 제품 정보 확인

계정, 제품 검색을 통해 쉽게 원하는 정보를 찾을 수 있다.

***

### a. 계정 정보 확인 방법

{% stepper %}
{% step %}
#### Easy Check 페이지에 접속해 검색창을 누른다.

<figure><img src="../.gitbook/assets/7-1. 계정 정보 확인(1).png" alt="" width="284"><figcaption></figcaption></figure>

> ℹ️ **홈에 나열된 계정 목록은 등록된 모든 계정을 표시하며, 목록의 항목을 눌러서 계정 상세 화면으로 이동할 수 있다.**
>
> <p align="center"><img src="../.gitbook/assets/7-1. 계정 정보 확인(홈 계정 리스트).png" alt="" data-size="original"></p>
{% endstep %}

{% step %}
#### 계정 정보를 입력한 후 목록에서 원하는 계정을 선택한다.

<figure><img src="../.gitbook/assets/7-1. 계정 정보 확인(2).png" alt="" width="295"><figcaption></figcaption></figure>

> ℹ️ **검색어를 입력하지 않으면 최근 조회한 계정 목록이 표시된다.**
>
> <p align="center"><img src="../.gitbook/assets/7-1. 계정 정보 확인(최근조회) (1).png" alt="" data-size="original"></p>
{% endstep %}

{% step %}
#### 계정 정보 상세에서 원하는 정보를 확인한다.

<figure><img src="../.gitbook/assets/7-1. 계정 정보 확인(3).png" alt="" width="270"><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

{% endstep %}

{% step %}

> <p align="center"><img src="../.gitbook/assets/7-1. 제품 정보 확인(최근조회).png" alt="" data-size="original"></p>
