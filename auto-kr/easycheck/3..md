# 3. 실시간 모니터링

## 7-3. 실시간 모니터링

실시간 모니터링 기능은 메인컨트롤러가 **연결된 상태(ONLINE)** 일 때 장비의 위치, 움직임 상태, 작업 진행 여부 등을 실시간으로 확인할 수 있는 화면이다.

{% hint style="danger" %}
안전 상의 이유로 주행 중에는 데이터 변경하지 않으며, 변경 시 반드시 고객과 실시간으로 소통하며 진행한다.
{% endhint %}

***

> **ℹ️ 접속 후 메인컨트롤러와 연결되지 않은 상태에서는 실시간 모니터링을 사용할 수 없으며, 5분 이상 미연결 상태가 지속되면 연결이 끊어진 것으로 간주한다.**
