# 8. 주행 기록 조회

## 7-8. 주행 기록 조회 <a href="#id-7-5" id="id-7-5"></a>

과거 주행한 세션을 날짜별로 조회하고, 각 주행의 상세 정보를 확인할 수 있는 기능이다.

***

### a. 주행 기록 조회 방법

{% stepper %}
{% step %}
#### 계정 상세 또는 메인컨트롤러 상세에서 \[모든 주행 세션 보기]를 누른다.

<figure><img src="../.gitbook/assets/7-7. 주행 기록 조회(1).png" alt="" width="563"><figcaption></figcaption></figure>

> ℹ️ **최근 주행한 세션인 경우, 최근 주행 세션 영역에서 해당 카드의 \[상세 정보 보기] 버튼을 눌러 바로 조회할 수 있다.**
{% endstep %}

{% step %}
#### 날짜를 선택하고 노출된 주행 세션 목록에서 원하는 카드의 \[상세 정보 보기] 버튼을 누른다.

<figure><img src="../.gitbook/assets/7-7. 주행 기록 조회(2).png" alt="" width="270"><figcaption></figcaption></figure>

> ℹ️ **주행 세션이 있는 날은 배경이 하늘색으로 표시된다.**
>
> <img src="../.gitbook/assets/7-7. 주행 기록 조회(주행 달력 설명).png" alt="" data-size="original">


{% endstep %}

{% step %}
#### 주행 상세 보기 화면으로 진입해 주행 기록 정보를 확인한다.

<figure><img src="../.gitbook/assets/7-7. 주행 기록 조회(3).png" alt="" width="270"><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
