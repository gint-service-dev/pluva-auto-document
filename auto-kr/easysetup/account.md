# 5. 계정 생성/확인

## 3-5. 계정 생성

자율주행에 사용되는 장비를 등록하는 과정이다.

***

### a. 인증 번호를 받은 경우

{% stepper %}
{% step %}
#### \[인증 요청]을 누른다.

<figure><img src="../.gitbook/assets/3-5. 계정생성-인증번호 받음 (1).png" alt="" width="256"><figcaption></figcaption></figure>

> **ℹ️ 주문서에서 작성한 정보가 자동으로 입력된다. (수정 가능)**
>
> **ℹ️ 휴대번호는 플루바서비스 이용을 위한 계정 인증 및 중요 알림 전송에 사용되므로 반드시 고객이 사용 중인 번호를 입력한다.**
>
> **ℹ️ 고객이 이미 플루바오토 회원일 경우 휴대번호 인증 요청 시 아래와 같은 메시지가 표시된다.**
>
> <img src="../.gitbook/assets/3-5. 계정생성-인증번호 받음-동일고객 존재 (1).png" alt="" data-size="original">
{% endstep %}

{% step %}
#### 고객의 휴대전화 메시지로 수신한 인증번호를 입력한다.

<figure><img src="../.gitbook/assets/3-5. 계정생성-인증번호 받음 (2).png" alt="" width="248"><figcaption></figcaption></figure>

> **ℹ️ 인증번호를 받지 못한 경우 \[**[**인증번호를 못받은 경우**](account.md#undefined-1)**🔗]를 참고해 가입을 진행한다.**
{% endstep %}

{% step %}
#### \[확인]을 누른다.

<figure><img src="../.gitbook/assets/3-5. 계정생성-인증번호 받음 (3).png" alt="" width="248"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### 휴대번호 인증 완료.  \[다음]을 누른다.

<figure><img src="../.gitbook/assets/3-5. 계정생성-인증번호 받음 (4).png" alt="" width="253"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### 휴대번호 인증 완료.  \[다음]을 누른다.

<figure><img src="../.gitbook/assets/3-5. 계정생성-인증번호 받음 (5).png" alt="" width="248"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### 계정 정보를 확인하고 \[완료]를 눌러 Easy Setup을 끝낸다.

<figure><img src="../.gitbook/assets/3-5. 계정생성-인증번호 받음 (6).png" alt="" width="248"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### Easy Setup 완료

<figure><img src="../.gitbook/assets/3-5. 계정생성-인증번호 받음 (7).png" alt="" width="248"><figcaption></figcaption></figure>

> **ℹ️&#x20;**<mark style="color:$primary;">**\[완료]**</mark>**&#x20;버튼을 누르면 Easy Setup이 완료된다. 설치티켓의 상태가 설치중 → 완료로 변경된다.**
>
> **ℹ️ 계정 생성 시 플루바오토 로그인에 사용되는 임시 비밀번호가 발급된다.**
>
> * **임시 비밀번호는 Easy Setup 페이지에서만 확인할 수 있다.**
> * **보안을 위해 계정 생성 직후 로그인하여 비밀번호를 변경하도록 안내한다.**
>
> **ℹ️&#x20;**<mark style="color:$primary;">**\[더보기]**</mark>**를 누르면 이지셋업에서 입력한 정보를 상세히 확인할 수 있다.**\
> **어드민의&#x20;**<mark style="color:$primary;">**설치티켓 상세**</mark>**&#x20;화면에서도 정보를 확인할 수 있다.**
{% endstep %}
{% endstepper %}

***

### b. 인증 번호를 못 받은 경우

{% hint style="danger" %}
**설치 담당자 권한으로 인증**\
핸드폰의 국제번호 수신 차단 등으로 인증번호 문자를 받지 못하는 경우, 설치 담당자가 대신 계정을 생성할 수 있는 예외 절차이다.\
단, 기본적인 인증 과정은 실 사용 고객 판별을 위해 반드시 필요하므로, <mark style="color:$danger;">**이**</mark> <mark style="color:$danger;">**기능은 불가피한 상황에서만 사용한다.**</mark> \ <mark style="color:$danger;">**계정 생성 및 인증에 대한 책임은 설치 담당자에게 있다.**</mark>
{% endhint %}

{% stepper %}
{% step %}
#### \[인증 요청]을 누른다.

<figure><img src="../.gitbook/assets/3-5. 계정생성-인증번호 못받음 (1).png" alt="" width="228"><figcaption></figcaption></figure>

> **ℹ️ 설치티켓 담당자 정보가 있을 경우 담당자 정보가 자동으로 입력된다.**
>
> **ℹ️ 담당자 정보 수정이 가능하며 담당자 정보 변경 시 설치티켓에 반영된다.**
{% endstep %}

{% step %}
#### \[인증 번호를 받지 못했나요?]을 누른다.

<figure><img src="../.gitbook/assets/3-5. 계정생성-인증번호 못받음 (2).png" alt="" width="226"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### 팝업창에서 \[담당자 권한으로 인증]을 누른다.

<figure><img src="../.gitbook/assets/3-5. 계정생성-인증번호 못받음 (3).png" alt="" width="226"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### 담당자 이름과 휴대번호를 입력한다.

<figure><img src="../.gitbook/assets/3-5. 계정생성-인증번호 못받음 (4).png" alt="" width="226"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### 아래 확인사항을 체크한 다음 \[휴대번호 인증]을 누른다.

<figure><img src="../.gitbook/assets/3-5. 계정생성-인증번호 못받음 (5).png" alt="" width="231"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### 고객의 이메일 주소를 입력하고 \[다음]을 누른다. (선택 입력 사항)

<figure><img src="../.gitbook/assets/3-5. 계정생성-인증번호 못받음 (6).png" alt="" width="248"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### 계정 정보를 확인하고 \[완료]를 눌러 Easy Setup을 끝낸다.

<figure><img src="../.gitbook/assets/3-5. 계정생성-인증번호 못받음 (7).png" alt="" width="248"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
#### Easy Setup 완료

<figure><img src="../.gitbook/assets/3-5. 계정생성-인증번호 못받음 (8).png" alt="" width="248"><figcaption></figcaption></figure>
