### <mark style="background-color:$primary;">b.설치 방법</mark>

{% columns fullWidth="false" %}
{% column width="58.333333333333336%" %}
#### 1. 트랙터에 올라가 메인 컨트롤러 판넬을 부착할 위치를 확인한다.

<figure><img src="../.gitbook/assets/메인컨트롤러 설치-설치(1).png" alt=""><figcaption></figcaption></figure>


{% endcolumn %}

{% column width="41.666666666666664%" %}
* <mark style="color:$primary;">보닛 훼손 방지를 위해 보닛의 가장자리 및 로더 암 부분을 밟고 올라간다.</mark>
* <mark style="color:$primary;">필요시 사다리를 이용한다.</mark>

> ⚠️  <mark style="color:$danger;">**주의사항**</mark>\
> 오름 시 추락에 주의한다. 필요한 경우 농기계를 타공할 수도 있다.
{% endcolumn %}
{% endcolumns %}





{% columns %}
{% column width="58.333333333333336%" %}
#### **2.** 트랙터 지붕의 기울기를 확인 후 부착할 위치에 이물질을 제거한다.

<figure><img src="../.gitbook/assets/메인컨트롤러 설치-설치(2).png" alt=""><figcaption></figcaption></figure>


{% endcolumn %}

{% column width="41.666666666666664%" %}
* <mark style="color:$primary;">세척액 및 세척용 솔, 걸레를 이용하여 이물질을 제거한다.</mark>
{% endcolumn %}
{% endcolumns %}





{% columns %}
{% column width="58.333333333333336%" %}
#### **3.** 판넬 뒤에 붙어있는 스티커를 제거 후 중앙에 맞게 부착한다.

<figure><img src="../.gitbook/assets/메인컨트롤러 설치-설치(3).png" alt=""><figcaption></figcaption></figure>


{% endcolumn %}

{% column width="41.666666666666664%" %}
* <mark style="color:$primary;">트랙터 지붕의 전방, 중앙에 맞춰 부착한다.</mark>

> ⚠️  <mark style="color:$danger;">**주의사항**</mark>\
> 한번 부착을 하면 재부착이 힘들기 때문에, 신중하게 부착한다.\
> 트랙터 선루프에 메인컨트롤러를 부착할 경우 자율주행 중 선루프 개방이 제한된다.
{% endcolumn %}
{% endcolumns %}





{% columns %}
{% column width="58.333333333333336%" %}
#### **4.** 메인 컨트롤러 보호 커버를 케이블 타이를 이용하여 결합한다.

<figure><img src="../.gitbook/assets/메인컨트롤러 설치-설치(4).png" alt=""><figcaption></figcaption></figure>
{% endcolumn %}

{% column width="41.666666666666664%" %}
* <mark style="color:$primary;">방향에 맞게 끼운 후 결합한다.</mark>
{% endcolumn %}
{% endcolumns %}





{% columns %}
{% column width="58.333333333333336%" %}
#### 5. 메인 컨트롤러를 부착한 판넬에 결합한다.

<figure><img src="../.gitbook/assets/메인컨트롤러 설치-설치(5).png" alt=""><figcaption></figcaption></figure>


{% endcolumn %}

{% column width="41.666666666666664%" %}
* <mark style="color:$primary;">메인 컨트롤러 연결 단자 위치의 자석부터 결합한다.</mark>

> ⚠️  <mark style="color:$danger;">**주의사항**</mark>\
> 부착 후 메인 컨트롤러가 중앙에 부착 되어 있는지 확인한다.
{% endcolumn %}
{% endcolumns %}





{% columns %}
{% column width="58.333333333333336%" %}
#### **6.** 메인 컨트롤러에 하네스를 결합한다.

<figure><img src="../.gitbook/assets/메인컨트롤러 설치-설치(6).png" alt=""><figcaption></figcaption></figure>


{% endcolumn %}

{% column width="41.666666666666664%" %}
* <mark style="color:$primary;">단자 위쪽에 볼록하게 튀어나온 홈을 확인 후 딸깍 소리가 나도록 결합한다.</mark>
{% endcolumn %}
{% endcolumns %}





{% columns %}
{% column width="58.333333333333336%" %}
#### **7.** 하네스 마운트를 이용하여 깔끔하게 정리한다.

<figure><img src="../.gitbook/assets/메인컨트롤러 설치-설치(7).png" alt=""><figcaption></figcaption></figure>


{% endcolumn %}

{% column width="41.666666666666664%" %}
* <mark style="color:$primary;">메인 컨트롤러가 탈 부착이 가능하도록 여유를 두고 마운트 및 케이블 타이로 깔끔하게 정리한다.</mark>

> ⚠️  <mark style="color:$danger;">**주의사항**</mark>\
> 하네스 정리 중 배기관 및 엔진열에 의해 파손되지 않도록 주의하며 정리한다.
{% endcolumn %}
{% endcolumns %}





{% columns %}
{% column width="58.333333333333336%" %}
#### **8.** 인터페이스 박스측면 2번 칸에 하네스를 연결한다.

<figure><img src="../.gitbook/assets/메인컨트롤러 설치-설치(8).png" alt=""><figcaption></figcaption></figure>


{% endcolumn %}

{% column width="41.666666666666664%" %}
* <mark style="color:$primary;">우측 앞단 2번 단자를 이용하여 연결한다.</mark>

> ⚠️  <mark style="color:$danger;">**주의사항**</mark>\
> 4번 단자에 연결 시 고장의 원인이 된다.
{% endcolumn %}
{% endcolumns %}
