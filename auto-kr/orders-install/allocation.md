# 2. 설치 담당자 배정

## 1-2. 설치 담당자 배정

설치를 진행할 담당자를 배정한다.\
담당자를 배정하지 않아도 설치를 진행할 수 있으며, 설치가 시작되면 담당자를 배정 및 변경할 수 없다.

***

{% hint style="info" %}
손가락으로 화면을 벌려 메뉴얼의 설명 이미지를 확대할 수 있다.
{% endhint %}

#### 1. 어드민 주문서 상세 화면에서 \[설치티켓 상세보기] 버튼을 눌러 티켓 상세 화면에 접속한다.

<figure><img src="../.gitbook/assets/1-2-1. 설치 담당자 배정.png" alt=""><figcaption></figcaption></figure>



#### 2. \[담당자 입력] 버튼을 클릭한다.

<figure><img src="../.gitbook/assets/1-2-2. 설치 담당자 배정.png" alt=""><figcaption></figcaption></figure>



#### 3. 담당자 이름과 전화번호를 입력하고 \[입력] 버튼을 클릭한다.

<figure><img src="../.gitbook/assets/1-2-3. 설치 담당자 배정.png" alt="" width="488"><figcaption></figcaption></figure>

> **ℹ️  담당자 정보는 선택 사항이므로 입력하지 않아도 설치를 진행할 수 있다. 설치 이력 관리가 필요한 경우 사용한다.**
>
> **ℹ️  설치가 시작되면 담당자 정보를 입력 및 수정할 수 없다.**
