# 1. 주문서 작성

## 1-1. 주문서 작성

플루바오토 구매 접수 시 주문서를 작성한다.\
주문서에 고객 정보, 설치 요청사항 등을 입력하여 주문과 설치 작업에 필요한 정보를 관리한다.

***

{% hint style="info" %}
손가락으로 화면을 벌려 메뉴얼의 설명 이미지를 확대할 수 있다.
{% endhint %}

#### 1. 플루바 오토 어드민 \[주문 관리] > \[주문서 등록/조회] 메뉴에서 \[주문서 작성] 버튼을 클릭한다.

<figure><img src="../.gitbook/assets/1-1. 주문서 작성.png" alt=""><figcaption></figcaption></figure>



#### 2. 플루바오토를 구매하는 사람의 정보를 입력한다.&#x20;

<figure><img src="../.gitbook/assets/1-2. 주문서 작성.png" alt=""><figcaption></figcaption></figure>



#### 3. \[고객 확인] 버튼을 눌러 신규/기존 고객 여부를 확인한다.

<figure><img src="../.gitbook/assets/1-3. 주문서 작성.png" alt=""><figcaption></figcaption></figure>

<details>

<summary>고객 확인 시 결과</summary>

*   구매 이력이 없는 고객 : "새로운 고객으로 등록됩니다" 표시

    <figure><img src="../.gitbook/assets/고객확인결과 새로운 고객.png" alt=""><figcaption></figcaption></figure>
*   구매 이력이 있는 고객:  "기존 고객으로 확인되었습니다." 표시

    <figure><img src="../.gitbook/assets/고객확인결과 기존고객.png" alt=""><figcaption></figcaption></figure>
*   주문자의 휴대번호가 기존 고객의 휴대번호와 동일한 경우 기존 고객 정보를 사용하거나 다른 휴대번호를 입력해야 한다.

    <figure><img src="../.gitbook/assets/유사고객 발견 모달.png" alt="" width="300"><figcaption></figcaption></figure>

</details>



#### 4. 플루바오토를 설치할 차량의 정보를 입력한다.&#x20;

<figure><img src="../.gitbook/assets/1-4. 주문서 작성 (1).png" alt=""><figcaption></figcaption></figure>

<details>

<summary>풀세트와 반세트란? </summary>

* 풀세트
  * 플루바오토의 완전한 구성품으로 메인컨트롤러, 오토스티어, 인터페이스박스, 원터치스위치를 포함한다.
  * 신규 고객이나, 여러 농기계를 동시에 사용하여 각각 독립적인 메인컨트롤러가 필요한 고객이 주로 주문한다.
* 반세트
  * 오토스티어, 인터페이스박스, 원터치스위치를 포함한다. (메인컨트롤러 미포함)\\
  * 농기계를 번갈아 사용하여 메인컨트롤러 하나를 여러 농기계에 옮겨가며 작업할 수 있는 고객이 주로 주문한다.
  * 반세트 주문 시 고객이 플루바오토를 사용하고 있는지(메인컨트롤러를 보유하고 있는지) 확인이 필요하다.
* 여러 세트를 구매하는 경우 \[세트 추가] 버튼을 눌러 주문서를 작성한다.
  * 예1. 트랙터 2대, 이앙기 1대에 모두 플루바오토를 설치하고 싶은 신규 고객 => \[풀세트 추가] 버튼을 눌러 총 3개의 풀세트를 주문한다.
  * 예2. 플루바오토를 트랙터에 작년 설치했는데 이번에 이앙기를 새로 구매하여 메인컨트롤러를 옮겨서 사용하려는 고객 => \[삭제] 버튼을 눌러 풀세트 양식을 삭제하고 \[반세트 추가] 버튼을 눌러 총 1개의 반세트를 주문한다.

</details>

<details>

<summary>제조사와 모델 리스트에 원하는 항목이 없을 경우</summary>

제조사와 모델 리스트에 원하는 항목이 없을 경우 체크박스를 클릭하여 직접 입력한다.

<figure><img src="../.gitbook/assets/풀세트와 반세트.png" alt=""><figcaption></figcaption></figure>

</details>



#### 5. 미리 고객에게 확인해두면 좋은 항목들을 체크리스트에서 확인하고 \[주문 완료] 버튼을 클릭한다.

<figure><img src="../.gitbook/assets/1-5. 주문서 작성.png" alt=""><figcaption></figcaption></figure>

> **ℹ️  체크리스트 항목을 미리 확인해두면 설치 당일 시간을 절약할 수 있다.**
>
> **ℹ️  항목에 체크하지 않아도 주문서를 작성할 수 있다. (필수 항목 아님)**
>
> **ℹ️  Easy Setup에서도 체크리스트와 동일한 내용을 확인할 수 있다.**
