# 퀵메뉴얼

### 🧾 관리자 계정 생성 방법

<details>

<summary>어드민 계정 발급 방법</summary>

1. **플루바 오토 제작사인 (주)긴트에게 아래 정보를 작성해 계정 발급을 요청**한다.
   1. 이름
   2. 이메일
   3. 대리점 명
2. 요청한 이메일로 **임시 비밀번호**를 수신한다.&#x20;
3. 수신한 **이메일(ID)과 임시 비밀번호**로 **어드민**과 **EasyCheck**에 로그인한다.
4. 첫번째 로그인 시 발급 받았던 임시 비밀번호를 변경한다.

</details>

#### 원인

> * **RTK 앱이 백그라운드에 있을 때 고객이 실수로 RTK 앱을 종료한다.**

> * **배터리 절약을 위해 안드로이드가 RTK 앱을 자동 방지 안내한다.**
> * **정기적으로 RTK 연결 상태를 확인한다.**

<details>

<summary><strong>주행 화면에서 자율주행을 할 수 없는 경우</strong></summary>

#### 원인

> 1. **차량제어 설정에서 지정한 주행 설정 값을 충족하지 않게 주행하는 경우**
>    * **자율주행 해제 조건을 벗어난 경우**
>      * **최저 속도 조건보다 저속으로 주행하거나 정지한 경우**
>      * **속도 조건보다 빠르게 주행한 경우**
>    * **자율주행 시작 조건을 벗어난 경우**
>      * **주행 속도가 설정 값보다 초과된 경우**
>      * **주행 각도가 설정 값보다 초과한 경우**
>      * **주행 거리가 설정 값보다 초과한 경우**
>    * **메인컨트롤러에 불량이 발생한 경우**

#### **조치 방법**

> 1. **차량제어 설정 값을 점검하고 필요에 따라 변경 (** [7..md](easycheck/7..md "mention") **참고)**
>    * **자율주행 해제 조건의 최저 속도, 속도 조건을 조정**
>    * **자율주행 시작 조건의 속도·거리·각도 조건을 조정**
> 2. **서비스 업체에 연락해 메인컨트롤러 교체 및 수리를 진행**<br>

</details>

<details>

<summary><strong>로그인이 되지 않는 경우</strong></summary>

#### 원인

> * **가입되지 않은 전화번호를 입력한 경우**
> * **비밀번호를 잘못 입력한 경우**

#### **조치 방법**

> * **올바른 전화번호가 아닐 수 있으니 입력한 전화번호를 확인한다.**
> * **비밀번호 재설정 후 로그인을 재시도한다.**

</details>

<details>

<summary><strong>골이 일직선으로 똑바르게 나오지 않는 경우</strong></summary>

#### 원인

> 1. **심한 작업기 유동**
> 2. **심한 트랙터 부하**
> 3. **센서 캘리브레이션 오차**

#### **조치 방법**

> 1. **3단 히치에 작업기를 단단히 고정해서 유격이 없도록 조치한다.**
> 2. **RPM을 올리거나 작업기를 올려 부하를 적게 받도록 조치한다.**
> 3. **주행 교정을 실시한다.**
>    1. **스티어링 재 교정을 실시한다. (** [autosteering.md](initialization/autosteering.md "mention") **참고)**
>    2. **IMU 신호의 노이즈가 발생할 수 있으므로 교정을 수행한다. (** [imu.md](initialization/imu.md "mention") **교정 참고)**

</details>

<details>

<summary><strong>등간격이 맞지 않는 경우</strong></summary>

#### 원인

> 1. **작업기가 트랙터의 정중앙에 위치하지 않음**
> 2. **작업기가 기울어져 편차 차이가 큼**
> 3. **작업기 치수 입력값이 올바르지 않음 (잘못 입력 시 등간격 편차 발생 가능)**
> 4. **메인 컨트롤러 설치가 정방향으로 되지 않음**
> 5. **(000) 이전 S/W 버전의 GPS 이상으로 작업지 이동 시 등간격 틀어짐**
> 6. **트랙터의 자체 수평제어 기능이 켜져있을 경우 < 현재 테스트 중**

#### **조치 방법**

> 1. **링크 길이 확인 후 정중앙에 맞게 작업기 위치를 조정한다.**
> 2. **수평계를 이용하여 작업기를 수평으로 맞춘다.**
> 3. **작업기 편차가 5cm 이내가 되도록 (좌·우 오프셋/편차, 센터 기준치 등)을 실측치와 대조하여 정확히 재입력한다. (** [#undefined-36](quick-manual.md#undefined-36 "mention") **참고)**
> 4. **메인 컨트롤러를 차량의 정중앙으로 조정하거나 플루바오토 앱에서 메인컨트롤러 설치 위치의 횡방향 설정 값을 조정한다. (** [19..md](auto-use/19..md "mention") **참고)**
> 5. **S/W 버전을 (000버전) 이후로 업데이트 한다.**
> 6. **수평제어 기능을 끄고 작업을 진행한다. < 현재 테스트 중**

</details>
