# ソフトウェアアップデート（OTA）

### ソフトウェアアップデート（OTA）

アップデートの有無は自動で確認され、更新が必要な際にお知らせします。

{% hint style="warning" %}
アップデートは、必ずLTEの受信状態が良好な場所や開けた場所（上空を遮るものがない場所）で行ってください。
{% endhint %}

{% hint style="info" %}
アップデートには、新機能の追加や安定性の改善、エラーの修正などが含まれています。
{% endhint %}

{% hint style="info" %}
内容によっては、すぐにインストールが必要な場合があります。
{% endhint %}

{% hint style="info" %}
アップデート完了後、再起動が必要な場合があります。
{% endhint %}

***

#### アップデートの種類についてのご案内

PLUVA iONのアップデートは、内容と重要度に応じて以下のようにご案内しております。

&#x20;![](<../../.gitbook/assets/icon-square-1 (1).svg>) **必須アップデート**

* 安全/安定性に影響を与えかねないため、必ずインストールしなければならないアップデートです。
  * 猶予期間が終了すると直ちにアップデートが実行されます。

&#x20;![](../../.gitbook/assets/icon-square-2.svg) **選択アップデート**

* 편利便性向上や内容変更を目的としたアップデートとして、ご希望のタイミングでインストールできます。
  *   インストールしなくても、引き続き現在のバージョンでご利用いただけ

      　　ます。

&#x20;![](../../.gitbook/assets/icon-square-3.svg) **緊急アップデート**

* 緊急トラブルを迅速に解決するため、直ちに実行すべきアップデートです。

***

#### アップデート方法のご案内

タブレットの電源を入れるとアップデートの通知が表示されますので、案内に従ってアップデートを行ってください。

{% stepper %}
{% step %}
**アップデートの案内を確認**

* アップデートを許可すると、アップデートが実行されます。
{% endstep %}

{% step %}
**アップデート進行中**

*   アップデートは、状況に応じて作業可能と作業不可に分かれます。

    * 一般アップデート（作業可能）

    　　　 ダウンロードやインストールされても既存のバージョンで作業が続けられます。<br>

    *   強制アップデート（作業不可）

        　　　 アップデートが完了するまで、製品の使用は制限されます。
{% endstep %}

{% step %}
**インストール完了および再起動のご案内**

* インストールが完了すると、新しいバージョンを適用するために再起動の案内が表示されます。
* 場合によっては再起動が2回必要となる可能性がありますので、画面の案内に従って進めてください。
{% endstep %}
