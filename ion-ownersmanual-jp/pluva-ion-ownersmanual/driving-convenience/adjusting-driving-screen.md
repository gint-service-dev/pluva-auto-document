# 走行画面の調整

### 走行画面の調整

***

#### 画面比率の調整機能

リアルタイムでの作業状態と走行画面の比率を適切に調整し、ご希望の作業画面が作れる機能です。

{% stepper %}
{% step %}
<img src="../../.gitbook/assets/adjusting-icon.svg" alt="" data-size="line"> 調整したい方向に画面比率の調整アイコンを動かすと、画面比率の調整ができます。

<figure><img src="../../.gitbook/assets/screen-ratio-adjustment-1.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

#### カメラビューをオンにする

作業機の状態をリアルタイムでモニタリングできるカメラビューです。\
後方の作業機を確認することで、安全かつ正確な作業をサポートします。

{% stepper %}
{% step %}
<img src="../../.gitbook/assets/ic_camera.svg" alt="" data-size="original"> カメラアイコンをクリックし、カメラビュー画面を表示してください。

<figure><img src="../../.gitbook/assets/turn-on-camera-view.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

#### カメラビュー画面のご案内 <a href="#camera-on" id="camera-on"></a>

<figure><img src="../../.gitbook/assets/camera-view-description.png" alt=""><figcaption></figcaption></figure>

&#x20;![](<../../.gitbook/assets/icon-square-1 (1).svg>) 拡大する

*   カメラ画面を走行画面の全画面表示に切り替えることができます。 「閉じるX」をクリックするとカメラ画面が終了されます。<br>

#### 経路全体を表示する

経路全体が確認できるよう画面の調整ができます。

{% stepper %}
{% step %}
<img src="../../.gitbook/assets/btn_full-view.svg" alt="" data-size="original"> 「圃場全体を表示する」アイコンをクリックします。

<figure><img src="../../.gitbook/assets/view-full-path-1.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
圃場全体に切り替わります。

<figure><img src="../../.gitbook/assets/view-full-path-2.png" alt=""><figcaption></figcaption></figure>

作業エリアを大きく表示したい場合 ![](../../.gitbook/assets/btn_small-view.svg) 「作業エリアの拡大」をクリックしてください。
{% endstep %}
{% endstepper %}

***

#### ミニ地図の設定

圃場全体が確認できるミニ地図を表示し、作業エリアの拡大モードとあわせて確認できます。

{% stepper %}
{% step %}
<img src="../../.gitbook/assets/drive-function-button.svg" alt="" data-size="original"> 「作業オプション」アイコンをクリックします。

<figure><img src="../../.gitbook/assets/mini-map-1.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
「ミニ地図」アイコンをクリックすると画面上にミニ地図が表示されます。

<figure><img src="../../.gitbook/assets/mini-map-2.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
