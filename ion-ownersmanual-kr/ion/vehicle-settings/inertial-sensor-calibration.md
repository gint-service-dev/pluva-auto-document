# 관성센서 보정

### 관성센서 보정

관성센서 보정은 차량의 기울기, 회전, 진동 데이터를 기준값으로 보정하여 자율주행 시 경로 추종 적확도를 높이는 절차입니다. 반드시 차량을 정차하고 시동을 끈 상태에서 진행합니다.

{% endstep %}

<figure><img src="../../.gitbook/assets/inertial-sensor-calibration-2.png" alt=""><figcaption></figcaption></figure>

{% step %}
