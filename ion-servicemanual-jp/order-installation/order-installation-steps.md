# 取付プロセス

### 開通-取付の案内

pluva iONは専門取付が必要です。アドミンの開通・取付プロセスに基づき、取付技術者が安全な取付を支援するとともに、取付履歴を管理します。

***

### 開通・取付の流れ

{% stepper %}
{% step %}
#### 取付プロセス&#x20;

* QRコードから簡単に開通できます。
* 専門的な教育を受けた取付担当者が直接訪問し取り付けを行います。
* 取付完了後には、お客様の状況に合わせた説明を実施します。

<figure><img src="../.gitbook/assets/Installation process sequence.png" alt=""><figcaption></figcaption></figure>


{% endstep %}

{% step %}
### 開通の準備

* 詳細内容は [開通の準備](order-registration.md)をご確認して下さい。


{% endstep %}

{% step %}
### QRコートの位置

* 詳細内容は [QRコートの位置](preparing-accounts.md)をご確認して下さい。


{% endstep %}

{% step %}
### 開通完了

* 詳細内容は [開通完了](product-registration.md)をご確認して下さい。


{% endstep %}

{% step %}
#### PLUVA iONの取り付け

* 車両に製品を取付けます。既存のハンドルを自動操舵専用ハンドルに交換し、各種センサーおよび装置を接続する作業が含まれます。
* 詳細内容は [PLUVA iONの取り付け](product-installation/)をご確認して下さい。


{% endstep %}

{% step %}
#### 簡単セットアップ&#x20;

{% hint style="info" %}
簡単セットアップについて?

お客様がタブレットをすぐにお使いいただけるよう、取付にて**初期環境をセットアップする段階**です。
{% endhint %}

* ネットワークや車両情報など、システム稼働に必要な**初期設定を完了させます**。設定が完了すると製品が有効化され、使用可能な状態になります。
* 詳細内容は [簡単セットアップ](quick-setup/)をご確認して下さい。


{% endstep %}

{% step %}
#### 取付完了の確認&#x20;

* 取付が完了した製品の試運転を行い、取付状態の点検および**誤作動・異常の有無を確認**します。問題が発見された場合は原因を特定し、必要に応じた対応を行います。


{% endstep %}
{% endstepper %}
