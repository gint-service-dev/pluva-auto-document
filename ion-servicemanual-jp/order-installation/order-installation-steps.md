# 注文および取り付けプロセスのご案内

pluva ionは、専門的な取り付け工程が必要な製品です。アドミン上の**注文から取り付けまでのプロセス**を 通じて、技術者による安全な取り付けをサポートし、取り付け履歴はアドミンにて厳重に管理されます。

***

### 注文および取り付けプロセス

{% stepper %}
{% step %}
**注文登録**

* アドミン上に注文情報を入力すると、ご購入いただいた製品の台数分の取り付けチケットが自動で発行されます。\
  取り付けチケットには、取り付けに必要な情報が含まれており、製品登録から開通、取り付け完了までの全てのプロセスを一元管理できます。
* 詳細は、[<mark style="color:$primary;">注文登録</mark>](order-registration.md)をご参照ください。
{% endstep %}

{% step %}
**お客様アカウントのご用意**

* サービスのご利用に必要なお客様のアカウントを予め準備してください。該当するアカウント情報に基づき、最適な自動操舵サービスのご提供が可能になります。
* 詳細は、[<mark style="color:$primary;">お客様アカウントのご用意</mark>](preparing-accounts.md)をご参照ください。
{% endstep %}

{% step %}
**取り付けチケットでの製品開通**

{% hint style="info" %}
取り付けチケットとは？

注文製品ごとの**取り付け状況を確認できる管理チケット**です。
{% endhint %}

* 取り付ける構成品の**シリアル番号と取り付けチケットを紐づけ、製品を開通**します。登録完了後には、製品のアクティベーションに必要な**開通キーが発行**されます。これにより、製品とお客様情報を紐づけた取り付け管理ができます。
* 詳細は、[<mark style="color:$primary;">製品登録</mark>](product-registration.md)をご参照ください。
{% endstep %}

{% step %}
**製品の取り付け**

* 車両に製品を取り付けます。既存のハンドルを自動操舵用のハンドルに交換し、各種センサーおよび機器を接続する作業が含まれます。
* 詳細は、[<mark style="color:$primary;">製品の取り付け</mark>](product-installation/)をご参照ください。
{% endstep %}

{% step %}
**簡単セットアップ**

{% hint style="info" %}
簡単セットアップとは？

取り付けの際にタブレットの**初期設定を実施**します。これにより、お客様は、導入後すぐにタブレットの運用ができます。
{% endhint %}

* 発行済みの**開通キーを**pluva ionのタブレットに**登録**し、ネットワーク及び車両情報など、システム作動に必要な**初期設定を完了**させます。設定完了後には、製品がアクティベートされ利用可能な状態になります。
* 詳細は、[<mark style="color:$primary;">簡単セットアップ</mark>](quick-setup/)をご参照ください。
{% endstep %}

{% step %}
**テスト走行**

* 取り付け完了後にはテスト走行を行います。正しく動作するかを確認し、**誤作動や異常の有無を確認**します。不具合が発生した際には、原因を調査し、適切な対応を実施します。
{% endstep %}

{% step %}
**お客様への操作説明**

* 製品の取り付けが完了されたら、安全な運用をサポートするための初期レクチャーを実施します。**主な機能や使用方法**についてご案内します。
* 詳細は、[<mark style="color:$primary;">使い方</mark>](/broken/pages/LJm8CV0ry1K8NPVMKZ2f)をご参照ください。
{% endstep %}

{% step %}
**取り付け完了**

* 全ての取り付け及び設定完了後には、取り付けチケットにて\[完了]を選択してください。取り付け完了状態がシステムに反映されます。
* 詳細は、[<mark style="color:$primary;">取り付け完了確認</mark>](installation-completed.md)をご参照ください。
{% endstep %}
{% endstepper %}
