#### GNSS受信機

GNSS受信機は、RTK高精度位置情報を基に農業機械の位置を精密に把握します。補正データを通じて位置補正が行われ、設定内容は作業環境に合わせて調整が必要な場合があります。

#### 電動ステアリングホイール

電動ステアリングホイールは、農業機械の既存のハンドルを取り外した後、ステアリング位置に取り付ける電動モーター式のステアリング機器です。生成された走行経路を基にステアリングを制御し、一定の　条件下でドライバーのステアリング作業をサポートします。

タブレットは、自動操舵システムのディスプレイ機器です。作業経路の設定や走行状況のモニタリング、システム制御及び設定を行い、リアルタイムで位置・走行情報・エラー状態を確認できます。

* 電動ステアリングホイール、GNSS受信機、タブレットなど、接続されたすべての機器の電源及び暗電流を一括でオン・オフします。

<div align="left"><figure><img src="../../.gitbook/assets/tablet-main-power-switch.png" alt="" width="150"><figcaption></figcaption></figure></div>

{% hint style="info" %}
タブレットの電源ON/OFF

タブレットを安定してご使用いただくため、以下の手順に従って電源のオン・オフを行てください。

* **電源を入れるとき：メイン電源スイッチをONに設定します。**
* **電源を切るとき：タブレットのスリープボタンを3\~5秒間長押しし、画面が消えたことを確認します。その後、メイン電源スイッチをOFFに設定してください。**
{% endhint %}

{% hint style="warning" %}
メイン電源スイッチのみで電源を切ると、タブレットがシャットダウンされない場合があります。必ずタブレットのスリープボタンで画面を消してから、メイン電源スイッチをOFFに設定してください。
{% endhint %}

{% hint style="warning" %}
タブレットを終了した後も、暗電流が流れるため、トラクターのバッテリーが上がる恐れがあります。長時間使用しない場合は、使用後に必ずメイン電源スイッチで電源を遮断してください。
{% endhint %}

### ハーネス

ハーネスは、タブレットやGNSS受信機などの各構成品を接続し、**電源および信号を送信**するための部品です。取り付けの際には、各装置が正常作動するよう、ハーネスを正確に接続してください。
