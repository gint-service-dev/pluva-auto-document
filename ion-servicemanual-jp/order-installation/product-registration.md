# 取り付けチケットでの製品開通

取り付けチケットに製品を登録し、開通キーを発行します。取り付けをスムーズに進めるため、予め開通手続きを進めることを推奨します。

{% hint style="info" %}
取り付けチケットとは？

注文した製品ごとの**取り付け状況を確認できるチケット**です。
{% endhint %}

***

#### 注文製品ごとの開通すべき構成品

ご注文いただいた製品ごとに、下記の構成品を用意してください。

1. **Pluva iON**

* 全ての構成品を登録します。
  * タブレット
  * GNSS受信機
  * 電動ステアリングホイール

2. **Pluva iONのエクスパンションキット（拡張キット）**

* タブレット以外の構成品を登録します。
  * GNSS受信機
  * 電動ステアリングホイール

3. **追加オプション**

* ワンタッチスイッチ

***

#### シリアル番号の登録（梱包番号）

製品登録は、製品に付着してあるQRコード（シリアル番号、または梱包番号）をスキャンして進めます。&#x20;

* 梱包番号（外箱のQRコード）を登録すると、構成品の**一括登録**ができます。

#### 取り付けチケットへのアクセス方法

{% stepper %}
{% step %}
[アドミンページ](https://gint-admin.pluva.kr/auth/operators/login)へログインします。

<figure><img src="../.gitbook/assets/enter-ticket-1 (1).png" alt="" width="348"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
取り付けチケット一覧にアクセスし、対象のチケットを選択します。

<figure><img src="../.gitbook/assets/enter-ticket-2 (1).png" alt="" width="336"><figcaption></figcaption></figure>

{% hint style="info" %}
左上の「詳細をみる」をタップし、「注文および取り付け管理」 > 「取り付けチケット一覧」にアクセスできます。

<img src="../.gitbook/assets/enter-ticket-list (1).png" alt="" data-size="original">
{% endhint %}
{% endstep %}
{% endstepper %}

***

#### 製品の開通方法

{% stepper %}
{% step %}
取り付けチケットから\[製品の開通開始]をタップします。

<figure><img src="../.gitbook/assets/enter-ticketdetail-1 (1).png" alt="" width="323"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
\[パッケージでまとめて開通する]をタップします。

<figure><img src="../.gitbook/assets/enter-ticketdetail-2 (1).png" alt="" width="320"><figcaption></figcaption></figure>

{% hint style="info" %}
各構成品の項目を選択すると、個別登録もできます。
{% endhint %}
{% endstep %}

{% step %}
梱包番号のQRコードをスキャンします。

<figure><img src="../.gitbook/assets/enter-ticketdetail-3 (1).png" alt="" width="270"><figcaption></figcaption></figure>

{% hint style="info" %}
カメラでコードを読み取れない場合は、入力欄をタップして梱包番号を直接入力します。

<img src="../.gitbook/assets/number-direct-input (1).png" alt="" data-size="original">
{% endhint %}
{% endstep %}

{% step %}
梱包番号を確認し、\[確認完了]をタップします。

<figure><img src="../.gitbook/assets/enter-ticketdetail-4 (1).png" alt="" width="315"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
スキャンが終わると、画面上に表示される製品登録のポップアップから\[開通完了]をタップします。

<figure><img src="../.gitbook/assets/enter-ticketdetail-5 (1).png" alt="" width="311"><figcaption></figcaption></figure>

{% hint style="info" %}
梱包番号（シリアル番号）が無効な場合、QRスキャン画面に戻ります。

<img src="../.gitbook/assets/invalid-ticket (1).png" alt="" data-size="original">
{% endhint %}

{% hint style="info" %}
追加オプションをご注文いただいた場合は、主な製品の開通後に追加オプションの開通を進めて下さい。全ての構成品の開通手続きを行ってこそ開通が完了します。

<img src="../.gitbook/assets/addoption (1).png" alt="" data-size="original">
{% endhint %}
{% endstep %}

{% step %}
製品の開通が完了します。

<figure><img src="../.gitbook/assets/enter-ticketdetail-6 (1).png" alt="" width="180"><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

{% hint style="info" %}
製品の開通が完了すると、取り付けチケットに登録された製品のシリアル番号や構成品の情報を確認できます。

<img src="../.gitbook/assets/enter-ticketdetail-7.png" alt="" data-size="original">
{% endhint %}
