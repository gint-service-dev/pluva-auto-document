# 作業機の追加 (작업기 추가)

### 作業機の追加(オプション)

作業に使う作業機を追加します。





***

#### 作業機の追加方法 (작업기 추가 방법)

{% stepper %}
{% step %}
**\[追加する]**&#x3092;押します。

<figure><img src="../../.gitbook/assets/add-worker-1 (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
別名、作業機のタイプなどを入力してか&#x3089;**\[確認]**&#x3092;押します。

<figure><img src="../../.gitbook/assets/add-worker-2.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
