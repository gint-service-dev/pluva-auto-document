# 圃場の設定

圃場の設定は、お客様が作業する農場、農場主、圃場、枕地を登録および設定するプロセスです。

圃場の設定は、以下のプロセスで行われます。

1. **農場の登録**

* 現在地をもとに初期設定されます。
*

    <figure><img src="../../.gitbook/assets/my-farm-default.png" alt=""><figcaption></figcaption></figure>

2. **農場主の登録**

* お客様アカウントをもとに初期設定されます。
*

    <figure><img src="../../.gitbook/assets/my-farm-owner-default.png" alt=""><figcaption></figcaption></figure>

3. **圃場の登録**

* 作業を開始するためには、圃場を登録する必要があります。
* 下記の[圃場の登録](farm-setting.md#field)をご参照ください。

4. **枕地の登録**

* 作業環境に合わせて枕地を登録します。
* 下記の[枕地](farm-setting.md#headland)をご参照ください。

***

### 圃場の登録 <a href="#field" id="field"></a>

**圃場とは？**

> 圃場は、実際の作業が行われる作業エリアのことです。地図から登録、または境界を直接描いて登録できます。

#### 「圃場の登録」機能へのアクセス

{% stepper %}
{% step %}
\[登録する]をタップします。

<figure><img src="../../.gitbook/assets/quick-add-field-1.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
「後で登録」を選択した場合、農場管理 > 圃場へ接続し、\[+]をタップして登録できます。

<img src="../../.gitbook/assets/quick-add-field-later.png" alt="" data-size="original">
{% endhint %}
{% endstep %}

{% step %}
\[確認]をクリックします。

<figure><img src="../../.gitbook/assets/field-registration-4.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

#### 地図から登録

{% stepper %}
{% step %}
「地図から登録」オプションを選択し、\[確認]をクリックします。

<figure><img src="../../.gitbook/assets/map-registration-1.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
地図から圃場を選択します。

<figure><img src="../../.gitbook/assets/map-registration-2.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
圃場登録の基本設定は\[地図から選択]になっています。 境界を直接作成するには、\[直接描く]をクリックします。
{% endhint %}
{% endstep %}

{% step %}
境界の生成後、\[進入口の設定]をクリックし、ご希望の場所を選択します。

<figure><img src="../../.gitbook/assets/map-registration-3.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
進入・退出口の位置設定ポップアップから\[同じ場所に設定]を選択します。

<figure><img src="../../.gitbook/assets/map-registration-4.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
\[退出口を別途設定]を選択した場合は、進入/退出口をそれぞれ指定する必要があります。

<img src="../../.gitbook/assets/select-path-entry.png" alt="" data-size="original">
{% endhint %}
{% endstep %}

{% step %}
進入・退出口の設定が完了したら圃場情報を入力し、\[登録]をクリックします。

<figure><img src="../../.gitbook/assets/map-registration-5.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
圃場の登録が完了します。

<figure><img src="../../.gitbook/assets/map-registration-6.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

#### 車両から登録

直接車両を運転し、圃場の境界を走行しながらポイントを設定する方法です。地図に表示されない圃場や境界が不明確な場合に適してます。

{% stepper %}
{% step %}
圃場の登録オプションから「車両から登録」を選択し、\[確認]をタップします。

<figure><img src="../../.gitbook/assets/field-add-vehicle-1.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
「車両から登録」画面にアクセスできます。車両を圃場の各ポイントへ移動させ、ボタンを押してポイントを登録します。

<figure><img src="../../.gitbook/assets/field-add-vehicle-2.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
設定直後に、該当するポイントの修正ボタンが表示されます。位置が不正確な場合は、修正ボタンを押して再設定してください。
{% endhint %}

{% hint style="warning" %}
各ポイントは順番通りに設定してください。任意に順番を変えて設定すると、境界が正しく生成されません。
{% endhint %}
{% endstep %}

{% step %}
Dポイント以上の設定が完了すると、車両を進入・退出口に移動させ、進入口と退出口の設定ボタンを押して登録します。

<figure><img src="../../.gitbook/assets/field-add-vehicle-3.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
退出口は、作業完了後に出ていく経路です。設定後にも修正できます。
{% endhint %}

{% hint style="info" %}
ポイントは最大6地点まで設定できます。
{% endhint %}
{% endstep %}

{% step %}
各ポイントと退出口の設定が完了したら、「圃場の登録」ボタンが表示されます。**圃場の登録**をタップします。

<figure><img src="../../.gitbook/assets/field-add-vehicle-4.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
圃場に関する情報入力画面から、圃場名、農場、作物などを入力します。入力後に**登録**をタップすると圃場の登録が完了します。

<figure><img src="../../.gitbook/assets/field-add-vehicle-5.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

### 枕地の登録 <a href="#headland" id="headland"></a>

**枕地とは？**

> 枕地は、圃場の端で安全に旋回したり、作業機を操作したりするために確保する余白エリアです。圃場の境界から設定した距離の内側に作成され、作業環境に合わせて幅を調整することができます。

#### 「枕地の登録」へのアクセス

{% stepper %}
{% step %}
My Farmから「枕地」を選択します。

<figure><img src="../../.gitbook/assets/quick-headland-registration-1.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
![](../../.gitbook/assets/add-button.svg) 「枕地の追加」ボタンをタップします。

<figure><img src="../../.gitbook/assets/quick-headland-registration-2.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
ご希望の設定オプションを選択し確認を押すと、枕地の作成に入ります。

<figure><img src="../../.gitbook/assets/quick-headland-registration-3.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
枕地の設定モーダルから、詳細設定を入力後\[確認]をタップします。

<figure><img src="../../.gitbook/assets/quick-headland-registration-4.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
枕地の作成が完了します。

<figure><img src="../../.gitbook/assets/quick-headland-registration-5.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
