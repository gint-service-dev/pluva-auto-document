# 農場設定 (작업지 설정)

### 農場設定 (작업지 설정)

農場設定はお客様がご使用する農場、農場主、圃場、枕地を登録・設定する段階です。  작업지 설정은 고객이 사용할 농장, 농장 소유자, 필드, 헤드랜드를 등록·설정하는 과정입니다.

作業地設定は下記の順で実施します。  작업지 설정은 아래 순서로 진행합니다.



***

#### **1.** 農場**登録 (농장등록)**

* 農場は、基本的に現在地を基準に登録されます。 현재 위치를 기준으로 농장 등록은 기본으로 생성 됩니다.

<figure><img src="../../.gitbook/assets/tablet 68 - Set up a workfield.png" alt=""><figcaption></figcaption></figure>



#### **2.** 農場主**登録 (농장 소유자 등록)**

* 農場主は、基本的にお客様のアカウントを基準に登録されます。 고객 계정을 기준으로 농장 소유자 등록은 기본으로 생성 됩니다.

<figure><img src="../../.gitbook/assets/tablet 69 - Set up a workfield.png" alt=""><figcaption></figcaption></figure>



#### **3. 圃場の登録** (**필드 등록)**

* 작업을 진행하려면 필드를 등록해야 합니다.
* 아래 [필드 등록](farm-setting.md#undefined-1)을 참고하세요.



#### **4. 枕地の生成 (헤드랜드 등록)**

* 작업 환경에 맞게 헤드랜드를 등록합니다.
* 아래 [헤드랜드](farm-setting.md#undefined-5)를 참고하세요.





***

#### 4. **枕地の生成 (헤드랜드 등록)**

> * 枕地は、圃場の端で安全にUターンしたり、作業機を操作するための空きスペースです。 헤드랜드는 필드 가장자리에서 안전하게 회전하거나 작업기를 조작할 수 있도록 확보하는 여유 구간입니다. 
> *   圃場の境界から設定した距離分だけ内側に
>
>     作られ、作業環境に合わせて幅(長さ)を調整できます。   필드 경계에서 설정한 거리만큼 안쪽으로 만들어지며, 작업 환경에 맞춰 폭(거리)을 조정할 수 있습니다.



{% stepper %}
{% step %}
My Farmの枕地を選択してください。  My Farm의 헤드랜드 항목을 누릅니다.

<figure><img src="../../.gitbook/assets/tablet 80 - headland-registration.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
![](../../.gitbook/assets/add-button.svg) 枕地の生成を選択してください。  헤드랜드 생성 버튼을 누릅니다.

<figure><img src="../../.gitbook/assets/tablet 81 - headland-registration.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
ご希望のオプションを選択し確認を押すと、枕地の生成が始まります。  원하는 설정 옵션을 선택하고 확인을 누르면 헤드랜드 생성이 시작됩니다.

<figure><img src="../../.gitbook/assets/tablet 82 - headland-registration.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
枕地の設定モーダルから詳細設定項目を入力し\[確認]を押して下さい。 헤드랜드 설정 모달에서 세부 설정을 입력 후 \[확인]을 누릅니다.

<figure><img src="../../.gitbook/assets/tablet 83 - headland-registration.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
枕地が生成されました。 헤드랜드 생성이 완료됩니다.

<figure><img src="../../.gitbook/assets/tablet 84 - headland-registration.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
