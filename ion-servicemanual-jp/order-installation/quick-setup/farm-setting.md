# 圃場の設定

圃場の設定は、お客様が作業する農場、農場主、圃場、枕地を登録および設定するプロセスです。

圃場の設定は、以下のプロセスで行われます。

1. **農場の登録**

* 現在地をもとに初期設定されます。
*

```
<figure><img src="../../.gitbook/assets/my-farm-default.png" alt=""><figcaption></figcaption></figure>
```

2. **農場主の登録**

* お客様アカウントをもとに初期設定されます。
*

```
<figure><img src="../../.gitbook/assets/my-farm-owner-default.png" alt=""><figcaption></figcaption></figure>
```

3. **圃場の登録**

* 作業を開始するためには、圃場を登録する必要があります。
* 下記の[圃場の登録](farm-setting.md#field)をご参照ください。

4. **枕地の登録**

* 作業環境に合わせて枕地を登録します。
* 下記の[枕地](farm-setting.md#headland)をご参照ください。

***

### 圃場の登録 <a href="#field" id="field"></a>

**圃場とは？**

> 圃場は、実際の作業が行われる作業エリアのことです。地図から登録、または境界を直接描いて登録できます。

#### 「圃場の登録」機能へのアクセス

{% stepper %}
{% step %}
\[登録する]をタップします。

<figure><img src="../../.gitbook/assets/quick-add-field-1.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
「後で登録」を選択した場合、農場管理 > 圃場へ接続し、\[+]をタップして登録できます。

<img src="../../.gitbook/assets/quick-add-field-later.png" alt="" data-size="original">
{% endhint %}
{% endstep %}

{% step %}
「地図から登録」を選択し\[確認]をタップします。

<figure><img src="../../.gitbook/assets/quick-add-field-2.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
地図上で圃場を選択します。

<figure><img src="../../.gitbook/assets/quick-add-field-3.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
圃場登録の初期設定は\[地図から登録]になっています。境界を直接作成するためには、\[直接描く]をタップしてください。
{% endhint %}
{% endstep %}

{% step %}
境界が作成されたら\[進入口の設定]をタップし、ご希望の位置を選択します。

<figure><img src="../../.gitbook/assets/quick-add-field-4.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
進入口・退出口の位置設定に関するポップアップから、\[同じ場所に設定]を選択します。

<figure><img src="../../.gitbook/assets/quick-add-field-5.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
進入・退出口の位置が異なる場合は、\[退出口を別途設定]をタップします。

<img src="../../.gitbook/assets/select-path-entry (1).png" alt="" data-size="original">
{% endhint %}
{% endstep %}

{% step %}
進入・退出口の設定が完了したら、圃場情報を入力し\[登録]をタップします。

<figure><img src="../../.gitbook/assets/quick-add-field-6.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
圃場登録が完了します。

<figure><img src="../../.gitbook/assets/quick-add-field-7.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

![](<../../.gitbook/assets/icon-square-4-4 (2).svg>) **戻る**

![](<../../.gitbook/assets/icon-square-6-6 (2).svg>) **終了する**

* 境界登録を完了します。進入・退出口を選択すると、このボタンが使用可能になります。

### 枕地の登録 <a href="#headland" id="headland"></a>

**枕地とは？**

> 枕地は、圃場の端で安全に旋回したり、作業機を操作したりするために確保する余白エリアです。圃場の境界から設定した距離の内側に作成され、作業環境に合わせて幅を調整することができます。

***

#### 「枕地の登録」へのアクセス

{% stepper %}
{% step %}
My Farmから「枕地」を選択します。

<figure><img src="../../.gitbook/assets/quick-headland-registration-1.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
![](<../../.gitbook/assets/add-button (1).svg>) 「枕地の追加」ボタンをタップします。

<figure><img src="../../.gitbook/assets/quick-headland-registration-2.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
ご希望の設定オプションを選択し確認を押すと、枕地の作成に入ります。

<figure><img src="../../.gitbook/assets/quick-headland-registration-3.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
枕地の設定モーダルから、詳細設定を入力後\[確認]をタップします。

<figure><img src="../../.gitbook/assets/quick-headland-registration-4.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
枕地の作成が完了します。

<figure><img src="../../.gitbook/assets/quick-headland-registration-5.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

#### 枕地全体

全方向に同一の枕地を適用します。スピーディーに設定を完了できます。

<div align="left"><figure><img src="../../.gitbook/assets/full-headland.png" alt="" width="225"><figcaption></figcaption></figure></div>

**枕地全体の設定モーダル**

<div align="left"><figure><img src="../../.gitbook/assets/full-headland-modal (1).png" alt="" width="218"><figcaption></figcaption></figure></div>

![](<../../.gitbook/assets/icon-square-1-1 (2).svg>) **圃場の選択**

* 枕地を適用する圃場を選択します。

![](<../../.gitbook/assets/icon-square-2-2 (2).svg>) **枕地名**

* 識別するための枕地名を入力します。

![](<../../.gitbook/assets/icon-square-3-3 (2).svg>) **枕地の長さ**

* 枕地の幅を数値入力で設定します。
  * ![](<../../.gitbook/assets/full-headland-street (1).png>)

![](<../../.gitbook/assets/icon-square-4-4 (2).svg>) **枕地の数**

* 枕地の周回数を設定します。
  * 枕地を1周に設定した場合\
    ![](<../../.gitbook/assets/full-headland-setting-one-headland (1).png>)
  * 枕地を2周に設定した場合\
    ![](<../../.gitbook/assets/full-headland-setting-two-headland (1).png>)

![](<../../.gitbook/assets/icon-square-5-5 (2).svg>) **圃場の境界オフセット**

* 数値入力で圃場境界を調整します。ON/OFFの切り替えが可能で
* +は外側、-は内側へ境界線が移動します。
  * プラス（+）で数値設定の場合\
    ![](<../../.gitbook/assets/full-headland-plus-when-setting (1).png>)
  * マイナス（-）で数値設定の場合\
    ![](<../../.gitbook/assets/full-headland-minus-when-setting (1).png>)

![](<../../.gitbook/assets/icon-square-6-6 (2).svg>) **作業機**

* 作業に使用する作業機を選択します。

![](<../../.gitbook/assets/icon-square-7 (1).svg>) **メモ**

* 追加情報をメモします。
