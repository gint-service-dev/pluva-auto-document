# GNSS受信機の接続確認

タブレットとGNSS受信機の接続状態を確認するステップです。

{% hint style="info" %}
設定値は、タブレットのGNSS受信機の設定メニューから確認および変更ができます。
{% endhint %}

***

#### GNSS受信機の接続確認

{% stepper %}
{% step %}
GNSS受信機の接続確認画面にアクセスします。画面が表示されると、接続確認が自動的に開始されます。

<figure><img src="../../.gitbook/assets/quick-setup-gnssconnect-1.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
\[次のステップへ]をタップすると、GNSS受信機の接続確認が完了します。

<figure><img src="../../.gitbook/assets/quick-setup-gnssconnect-2.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
接続に失敗した場合は、受信機側面のLEDステータスを確認のうえ「再試行」をタップし、もう一度実行してください。

<img src="../../.gitbook/assets/quick-setup-gnssmisconnect.png" alt="" data-size="original">
{% endhint %}
{% endstep %}
{% endstepper %}
