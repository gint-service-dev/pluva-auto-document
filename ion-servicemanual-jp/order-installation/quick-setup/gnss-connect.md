# GNSS受信機の接続確認 (GNSS 수신기 연결 확인)

### GNSS受信機の接続確認

タブレットとGNSS受信機との接続状態を確認する段階です。

{% hint style="info" %}
設定値は、タブレットの [GNSS 수신기 위치 설정](vehicle-calibration.md#gnss)から確認および変更が可能です。
{% endhint %}

***

#### GNSS受信機の接続確認

{% stepper %}
{% step %}
GNSS受信機の接続確認にアクセスします。アクセスすると、自動で接続確認が行われます。

<figure><img src="../../.gitbook/assets/tablet 12 - Check GNSS receiver connection.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
接続が確認できたら\[次のステップへ]を押しRTK設定へアクセスします。

<figure><img src="../../.gitbook/assets/tablet 13 - Check GNSS receiver connection.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
