# RTK入力

### RTK入力







***

#### RTK入力(RTK直接受信する接続方法)

{% hint style="info" icon="circle-info" %}
RTKの手動接続は、タブレットからRTKサービスに直接接続し、補正信号を受信する方法です。

* イチミルなどのLTEのRTK-GPSを使用する場合、タブレットからRTKのデータ設定を行います。
* 予めRTK-GPSのアカウントを発行するよう、お客様へ案内してください。お客様がアカウント情報をお忘れの場合は、RTK-GPSシステムを申請した際に記載したメールアドレスからログインし、登録した際に受信したメールを確認するよう案内してください。.
* 基地局以外の全ての項目は、RTK-GPSの事業者が提供した情報をそのまま入力する必要があります。アルファベットの大文字や小文字を正確に入力し、不要なスペースは入れないようにしてください。
{% endhint %}



{% stepper %}
{% step %}
RTKの手動接続を選択&#x3057;**\[確認]**&#x3092;押して下さい。

<figure><img src="../../.gitbook/assets/tablet 14 - RTK.png" alt=""><figcaption></figcaption></figure>


{% endstep %}

{% step %}
基地局/サーバー情報を入力してか&#x3089;**\[接続]**&#x3092;押してください。

<figure><img src="../../.gitbook/assets/tablet 15 - RTK.png" alt=""><figcaption></figcaption></figure>


{% endstep %}

#### RTK入力(RTK Bluetooth接続方法)

{% hint style="info" icon="circle-info" %}
RTKのBluetooth接続は、スマートフォンのRTKアプリとBluetoothをペアリングし、基地局(RTK)の補正信号をGNSS受信機へ送信する設定です。

* PLUVA iONが正確な自動操舵をするために、作業中には RTK-GPSのアプリが継続して作動中である必要があります。RTKに接続できない場合は、まずRTK-GPSアプリが起動中であることをご確認下さい。
{% endhint %}

{% stepper %}
{% step %}
RTKのBluetooth接続を選択&#x3057;**\[確認]**&#x3092;押して下さい。

<figure><img src="../../.gitbook/assets/tablet 17 - RTK Bluetooth.png" alt=""><figcaption></figcaption></figure>


{% endstep %}

{% step %}
下記のプロセスに従って外部のBluetoothアプリ(RTK-GPSなど)からRTKのBluetooth接続設定を進めてください。

* &#x20;RTK – GPS アプリがインストールされたスマートフォンのBluetooth(ブルートゥース)を起動させます。
* スマートフォンorタブレットからRTK-GPSアプリを実行してください。
* アプリの設定から基地局の信号入力方法をBluetoothにしてください。
* 使用可能なデバイスリストからPLUVA iONを選択してください。
* パスワードを入力し、ペアリングを完了してください。
*   アプリ、またはスマートフォンorタブレットのBluetooth画面から接続済みであることを確認して下さい。


{% endstep %}

{% step %}
RTKのBluetooth設定が完了ます。**\[次のステップへ]**&#x3092;押すと車両の追加にアクセスします。

<figure><img src="../../.gitbook/assets/tablet 18 - RTK Bluetooth.png" alt=""><figcaption></figcaption></figure>


{% endstep %}
{% endstepper %}
