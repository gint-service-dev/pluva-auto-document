# ソフトウェアアップデート（OTA）

#### ソフトウェアアップデート（OTA）とは？

常に最新のパフォーマンスと安定した使用環境を提供するため、機器が自動的に最新バージョンを確認しアップデートを行います。

{% hint style="info" %}
最新のアップデートが必要な場合のみ、ソフトウェア設定画面が表示されます。
{% endhint %}

{% hint style="warning" %}
ソフトウェアアップデート（OTA）を行うと、約**300MB～500MB**のデータ通信量が発生します。お客様のデータ料金プランを予め確認のうえ、データが足りない場合は、Wi-Fiやテザリングを用意してください。
{% endhint %}

***

{% stepper %}
{% step %}
\[アップデート]をタップします。

<figure><img src="../../.gitbook/assets/Quick setup_OTA-1.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
ソフトウェアアップデートは、Wi-Fi環境で行うことを推奨します。
{% endhint %}

{% hint style="info" %}
ネットワークが「セルラー」に設定してある場合は、\[ネットワーク変更]から切り替えることができます。
{% endhint %}
{% endstep %}

{% step %}
アップデートが開始されます。

<figure><img src="../../.gitbook/assets/Quick setup_OTA-2.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
アップデートが完了したら、\[再起動]をタップします。

<figure><img src="../../.gitbook/assets/Quick setup_OTA-3.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
再起動後、タブレットに再度接続し\[次のステップへ]をタップすると、ソフトウェアアップデートが完了します。

<figure><img src="../../.gitbook/assets/Quick setup_OTA-4.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
