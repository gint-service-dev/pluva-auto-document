# 製品管理

製品管理は、各部署で担当する製品を一元管理するメニューです。管理者の所属部署に登録された製品のみが表示され、各製品の開通状況やソフトウェアのバージョン、関連した履歴を確認できます。

***

### アクセス方法

{% stepper %}
{% step %}
サイドバーから\[製品管理]をタップし、ご希望の項目を選択します。

<figure><img src="../.gitbook/assets/product-management-menu-open-1.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
製品リストの詳細へアクセスできます。

<figure><img src="../.gitbook/assets/product-management-menu-select-1.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

***

### 製品の詳細画面に関するご案内

#### PC環境

<figure><img src="../.gitbook/assets/product-management-detail-pc-1.png" alt=""><figcaption></figcaption></figure>

![](<../.gitbook/assets/icon-square-1 (1).svg>) **ステータス**

* 区分：開通、未開通

{% hint style="info" %}
取り付けチケットから、該当する製品が開通状態かどうかが表示されます。

ステータスは直接変更できません。取り付け完了時に自動で反映されます。
{% endhint %}

![](<../.gitbook/assets/icon-square-2 (1).svg>) **用途**

* 区分：販売、サービス、テスト

![](<../.gitbook/assets/icon-square-3 (1).svg>) **担当部署**

![](<../.gitbook/assets/icon-square-4 (1).svg>) **製造日**

![](../.gitbook/assets/icon-square-5.svg) **タブレットのシリアル番号**

![](../.gitbook/assets/icon-square-6.svg) **現在のソフトウェアバージョン**

* 現在の製品のアップデートバージョン

![](../.gitbook/assets/icon-square-7.svg) **次のソフトウェアバージョン**

* アップデート予定のバージョン（予定としているアップデートがない場合は未表示）

![](../.gitbook/assets/icon-square-8.svg) **メモ**

* 最新の作成者の名前が表示されます。メモは削除・修正でき、作成していない場合は、作成のボタンが表示されます。

![](../.gitbook/assets/icon-square-9.svg) **開通情報**

* 所有者の情報、開通日、Simカードの情報が表示されます。

{% hint style="info" %}
開通日は、取り付けチケットの完了処理日と同じ日付になります。
{% endhint %}

![](../.gitbook/assets/icon-square-10.svg) **アップデート履歴**

* ソフトウェアがアップデートされた日付とバージョンの情報が確認できます。
