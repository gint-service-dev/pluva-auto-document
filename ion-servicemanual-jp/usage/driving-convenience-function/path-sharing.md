# 경로 공유

공유 코드로 저장한 경로를 다른 사용자와 주고받을 수 있습니다. 또한 GPX/ISOXML 파일로도 경로를 가져올 수 있습니다.

***

**경로 공유 진입**

![](https://servicemanual.pluva.io/assets/ion/kr/drive-function-button.svg) \[작업] 버튼을 누릅니다.

![](https://servicemanual.pluva.io/assets/ion/kr/path-sharing-explanation-1.png)

경로 불러오기를 누릅니다.

![](https://servicemanual.pluva.io/assets/ion/kr/path-sharing-explanation-2.png)

\[경로 공유] 버튼을 누릅니다.

![](https://servicemanual.pluva.io/assets/ion/kr/path-sharing-explanation-3.png)

원하는 옵션을 선택합니다.

![](https://servicemanual.pluva.io/assets/ion/kr/path-sharing-explanation-4.png)

***

**경로 공유하기**

\[공유하기] 버튼을 누릅니다.

![](https://servicemanual.pluva.io/assets/ion/kr/how-to-share-1.png)

경로 코드를 상대방에게 전달해 경로 공유하기를 완료합니다.

![](https://servicemanual.pluva.io/assets/ion/kr/how-to-share-2.png)

***

**경로 공유받기**

\[공유받기] 버튼을 누릅니다.

![](https://servicemanual.pluva.io/assets/ion/kr/get-path-sharing-1.png)

공유받을 경로 코드를 입력 후 \[확인]을 누릅니다.

![](https://servicemanual.pluva.io/assets/ion/kr/get-path-sharing-2.png)

파일 가져오기를 선택한 경우 파일 형식을 선택해 경로를 가져옵니다.

![](https://servicemanual.pluva.io/assets/ion/kr/import-file.png)

경로 공유받기 완료됩니다.

![](https://servicemanual.pluva.io/assets/ion/kr/get-path-sharing-3.png)
