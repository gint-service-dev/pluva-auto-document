# 農場主の管理

### 農場主の管理

農場主の管理では、所有者情報を追加・修正・削除できます。 所有者ごとに登録された圃場の現状をも一緒に確認することができます。

***

#### 農場主の管理へアクセス

{% stepper %}
{% step %}
![](../../.gitbook/assets/ic_menu.svg) \[メニュー一覧のアイコン]をクリックします。

<figure><img src="../../.gitbook/assets/farm-owner-management-1.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
My Farmの農場管理の\[農場主]タブをクリックすると、農場主の管理へアクセスできます。

<figure><img src="../../.gitbook/assets/farm-owner-management-2.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

***

#### 農場主の追加

{% stepper %}
{% step %}
![](../../.gitbook/assets/ic_menu.svg) 農場主の追加ボタンを押します。

<figure><img src="../../.gitbook/assets/add-farm-owner-1.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
\[農場主の追加]ポップアップ上に内容を入力し、\[追加]ボタンを押します。

<figure><img src="../../.gitbook/assets/add-farm-owner-2.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
農場主の追加が完了します。

<figure><img src="../../.gitbook/assets/add-farm-owner-3.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

***

#### 農場主の修正

{% stepper %}
{% step %}
ご希望の農場主項目の ![](../../.gitbook/assets/ic_more.svg) アイコンをクリックします。

<figure><img src="../../.gitbook/assets/modifying-farm-owners-1.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
\[修正]をクリックします。

<figure><img src="../../.gitbook/assets/modifying-farm-owners-2.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
内容を修正し、\[修正完了]をクリックします。

<figure><img src="../../.gitbook/assets/modifying-farm-owners-3.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
修正が完了します。

<figure><img src="../../.gitbook/assets/modifying-farm-owners-4.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

***

#### 農場主の削除

{% stepper %}
{% step %}
ご希望の農場主項目の ![](../../.gitbook/assets/ic_more.svg) アイコンをクリックします。

<figure><img src="../../.gitbook/assets/delete-farm-owner-1.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
\[削除]をクリックします。

<figure><img src="../../.gitbook/assets/delete-farm-owner-2.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
\[農場主の削除]ポップアップから\[削除] をクリックします。

<figure><img src="../../.gitbook/assets/delete-farm-owner-3.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
農場主の削除が完了します。

<figure><img src="../../.gitbook/assets/delete-farm-owner-4.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
