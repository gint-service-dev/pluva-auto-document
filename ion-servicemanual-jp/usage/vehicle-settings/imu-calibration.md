# ロール／ピッチ／ヨー補正

ロール／ピッチ／ヨー補正は、傾斜地や不整地でも車両が正確な方向と姿勢を維持するために、GNSS受信機の基準値を調整する機能です。取り付け状態によって傾きが大きい場合は、値を入力して補正することができます。

{% stepper %}
{% step %}
![](<../../.gitbook/assets/ic_vehicle (1).svg>) \[車両]をタップします。

<figure><img src="../../.gitbook/assets/imu-calibration-1 (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
車両補正をタップしてください。

<figure><img src="../../.gitbook/assets/imu-calibration-2 (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
ロール、ピッチ、ヨー測定をタップするとアクセスできます。

<figure><img src="../../.gitbook/assets/imu-calibration-3 (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
ご希望の寸法を入力し、\[確認]を押すと適用されます。

<figure><img src="../../.gitbook/assets/imu-calibration-4 (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

***

#### ロール、ピッチ、ヨー 補正のご案内

<div align="left"><figure><img src="../../.gitbook/assets/imu-calibration-screen (1).png" alt=""><figcaption></figcaption></figure></div>

![](<../../.gitbook/assets/icon-square-1-1 (2).svg>) **ROll ロール**

* GNSS受信機が左右方向に傾いている角度を意味します。

![](<../../.gitbook/assets/icon-square-2-2 (2).svg>) **Pitch ピッチ**

* GNSS受信機が左右方向に傾いている角度を意味します。

![](<../../.gitbook/assets/icon-square-3-3 (2).svg>) **Yaw ヨー**

* GNSS受信機が垂直軸を基準に回転した角度を意味します。
