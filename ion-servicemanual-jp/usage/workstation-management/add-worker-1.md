# 作業機の追加

### 作業機の追加

作業に使用する作業機を登録し、作業機の偏差などの設定値も合わせて入力することができます。

***

#### 作業機の追加方法

{% stepper %}
{% step %}
![](../../.gitbook/assets/ic_worker.svg) \[作業機]をタップします。

<figure><img src="../../.gitbook/assets/add-worker-1.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
作業機リストへのアクセスができます。

<figure><img src="../../.gitbook/assets/add Worker-2.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
![](../../.gitbook/assets/add-button.svg) 作業機の追加ボタンをタップします。

<figure><img src="../../.gitbook/assets/add-worker-3.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
作業機のタイプや幅などの詳細情報を入力し、確認を押します。

<figure><img src="../../.gitbook/assets/add-worker-4.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
作業機の追加が完了します。

<figure><img src="../../.gitbook/assets/add-worker-5.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

***

#### 作業機の追加項目のご案内

<div align="left"><figure><img src="../../.gitbook/assets/add-worker-screen.png" alt="" width="208"><figcaption></figcaption></figure></div>

&#x20;![](../../.gitbook/assets/icon-square-1-1.svg) 作業機タイプ

* 作業機タイプを選択します。

&#x20;![](../../.gitbook/assets/icon-square-2-2.svg) 作業機の幅

* 作業機の幅を入力します。

&#x20;![](../../.gitbook/assets/icon-square-3-3.svg) 畝間

* 畝間を入力します。

&#x20;![](../../.gitbook/assets/icon-square-4-4.svg) リンク -作業機の距離

* リンクと作業機の間の距離を入力します。

&#x20;![](../../.gitbook/assets/icon-square-5-5.svg) 作業機の偏差

* 作業機の偏差を入力します。

&#x20;![](../../.gitbook/assets/icon-square-6-6.svg) 作業機偏差の計算

* 3列を走行してライン間隔の値を入力すると、作業機の偏差が自動で計算されます。
* 画像で示された方向に沿って3本のラインを走行し、走行ライン間の距離を測定して入力してください。<br>
*

    <figure><img src="../../.gitbook/assets/worker-deviation-calculation.png" alt=""><figcaption></figcaption></figure>

    * S1：一本目と二本目の間の距離
    * S2：二本目と三本目の間の距離
    * 画面上部のトグルから、左方向または右方向を選択できます。\
      ![](../../.gitbook/assets/toggle-on-off.png)
