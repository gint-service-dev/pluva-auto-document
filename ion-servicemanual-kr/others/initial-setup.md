# 고객 불편사항 대응 방법

### 빠른 문제 해결

제품을 사용하기 전에 확인해야 하는 기본 설정과 사전 안내 사항을 안내합니다.

***

#### 1) 어드민 계정이 없음

* **원인:**
  1. 어드민 계정 미발급
* **조치 방법:**
  1. 어드민 계정 요청 (필수 항목)
     1. &#x20;이름
     2. 이메일
     3. 대리점명

#### 2) 플루바 회원가입 페이지에 접속이 되지 않음

* **원인:**
  1. 홈페이지 주소 불량
* **조치 방법:**
  1. 홈페이지 주소 확인

#### 3) 플루바 회원가입이 되지 않음

* **원인:**
  1. 플루바 등록 회원
  2. 휴대폰 번호 불량
* **조치 방법:**
  1. PW 찾기 진행
  2. 휴대폰 번호 확인<br>

#### 4) 회원가입 중 인증번호 미 수신

* **원인:**
  1. 휴대폰 번호 불량
  2. 스팸문자함에 수신
  3. 네트워크 수신 불량
  4. 해외발신 차단
* **조치 방법:**
  1. 휴대폰 번호 확인
  2. 스팸문자함 확인
  3. 네트워크 확인 후 연결
  4. 해외 발신 차단 해제<br>

#### 5) 포장 박스에 QR코드 떨어짐

* **원인:**
  1. 배송중 QR코드 스티커 떨어짐 발생
* **조치 방법:**
  1. 제품별 QR코드 스캔<br>

#### 6) QR코드가 스캔되지 않음

* **원인:**
  1. QR코드와 거리가 멀어 인식 불량
  2. QR코드가 여러장 찍혀 인식 불량
  3. QR코드 인쇄 품질 불량
  4. 카메라 초점 불량
* **조치 방법:**
  1. 화면을 확대하여 QR코드 스캔
  2. 화면을 확대해 1장만 QR코드 스캔
  3. 직접 입력을 눌러 제품 등록
  4. 카메라 렌즈를 세척 후 재 촬영

#### 7) 기존 핸들이 탈거되지 않음

* **원인:**
  1. 기존 핸들에 쇳가루로 인해 고착
* **조치 방법:**
  1. 방청 윤활제를 이용하여 탈거
  2. 망치로 살살 옆을 쳐서 탈거&#x20;
  3. 3발 풀러를 이용하여 탈거
     1. 3발 풀러 이용시 조향축 파손 주의&#x20;

#### 8) 스플라인이 맞지 않음

* **원인:**
  1. 스플라인 종류 불량
  2. 스플라인 불량
* **조치 방법:**
  1. 장비에 맞는 스플라인 사용
  2. (주) GINT에 스플라인 요청&#x20;

#### 9) 프론트 브라켓 고정 불량

* **원인:**
  1. &#x20;규격에 맞지 않은 프론트 브라켓 사용
  2. M5x10 볼트 조임 불량
* **조치 방법:**
  1. A,B,기본브라켓 불량 시 U볼트 브라켓 사용
  2. M5x10볼트를 단단히 조임

#### 10) GNSS 수신기가 계속 앞으로 기움

* **원인:**
  1. M8x25볼트 조임 토크 불량
* **조치 방법:**
  1. GNSS 수신기 M8x25볼트 조임 시 10Nm 이상으로 조임 필요

#### 11) 태블릿 전원 이 들어오지 않음

* **원인:**
  1. 전원 스위치 불량
  2. 하네스 퓨즈 불량
  3. 트랙터 배터리 전압 불량
  4. 태블릿, 전원 스위치 단자 체결 불량
  5. 하네스 불량
  6. 태블릿 불량
* **조치 방법:**
  1. 전원 스위치에 불이 들어오는 지 확인 후 교환
  2. (+) 단자쪽 퓨즈함을 열어 퓨즈 단선 확인
  3. 트랙터 배터리 확인 후 12V 이상인지 확인
  4. 태블릿 및 전원 스위치  단자 재 결합
  5. 하네스 교환
  6. 태블릿 교환

#### 12) 태블릿에 인터넷이 연결 되지 않음

* **원인:**
  1. USIM 불량
  2. Wi-Fi 수신 불량
  3. 테더링 데이터 불량
* **조치 방법:**
  1. USIM 확인 후 수동입력
  2. Wi-Fi 확인
  3. 테더링기기 데이터 확인

#### 13) 태블릿에 로그인이 되지 않음

* **원인:**
  1. 플루바 회원가입이 되어 있지 않음
  2. ID 입력 불량
  3. PW 입력 불량
  4. 네트워크 연결 불량
* **조치 방법:**
  1. 플루바 회원가입 진행
  2. ID 찾기 진행
  3. PW 찾기 진행
  4. 네트워크 확인 후 연결

#### 14) 개통키가 인증되지 않음

* **원인:**
  1. 개통키 오입력
  2. 네트워크 불량
* **조치 방법:**
  1. 개통키 확인 후 재입력&#x20;
  2. 네트워크 확인 후 연결

#### 15) GNSS 수신기 연결 불량

* **원인:**
  1. GNSS 수신기 단자 체결 불량
  2. 하네스 불량
  3. GNSS 수신기 불량
* **조치 방법:**
  1. GNSS 수신기 단자 재결합
  2. 하네스 교환
  3. GNSS 수신기 교환

#### 16) RTK 직접 수신 연결이 되지 않음

* **원인:**
  1. RTK 계정이 없음
  2. RTK 정보 입력 오류
* **조치 방법:**
  1. 계정 가입 진행
  2. RTK 입력 정보 확인 후 수정
     1. 소문자 대문자 구분 필요

#### 17) RTK 블루투스 연결이 되지 않음

* **원인:**
  1. 외부 앱의 블루투스 설정 불량
  2. 외부 앱을 실행하는 스마트폰의 배터리 부족
* **조치 방법:**
  1. 외부 앱에서  블루투스 페어링 여부 확인
  2. 스마트폰의 배터리 충전

#### 18) 오토 스티어링 보정이 되지 않음

* **원인:**
  1. 측정 속도 불량
  2. 필지 불량
  3. 네트워크 불량
* **조치 방법:**
  1. 측정에 필요한 속도를 맞춰 재 측정
  2. 평평한 필지에서 재 측정
  3. 네트워크 확인 후 연결

#### 19) 자율주행 버튼이 생성되지 않음

* **원인:**
  1. 경로가 생성이 되지 않음
  2. GPS 헤딩 보정이 되지 않음
* **조치 방법:**
  1. 자율주행 경로 생성
  2. 3kph 속도 이상으로 주행

#### 20) 자율주행 버튼이 눌리지 않음

* **원인:**
  1. GPS 헤딩 보정이 되지 않음
  2. 거리 오차 발생
  3. 진입 각도 오차 발생
* **조치 방법:**
  1. 3kph 이상으로 주행
  2. 2m 이내에서 자율주행 버튼 클릭
  3. 40° 이내에서 자율주행 버튼 클릭

#### 21) 유턴이 작동하지 않음

* **원인:**
  1. 유턴 모드 비활성화
  2. 헤드랜드 설정되어 있지 않음
* **조치 방법:**
  1. 하단 작업-> 유턴모드 -> 유턴 토글을 눌러 활성화
  2. 헤드랜드 설정 방법 확인 후 설정 진행

#### 22) 경로 공유가 되지 않음

* **원인:**
  1. 태블릿 전원이 꺼져있음
  2. 네트워크 불량
  3. 공유 코드 오류
* **조치 방법:**
  1. 공유 할 태블릿과 받을 태블릿 전원 ON
  2. 네트워크 확인 후 연결

#### 23) 자율주행 중 골이 반듯하게 나오지 않음

* **원인:**
  1. 작업기 유동 발생
  2. 자동 수평모드 ON 상태
  3. 주행 중 로더 및 작업기 조작
  4. 프론트 브라켓 고정력 불량
* **조치 방법:**
  1. 작업기가 유동이 없도록 체크체인 고정
  2. 자동 수평모드 OFF
  3. 주행 중 로더 및 작업기 조절 최대한 지양
  4. 프론트 브라켓 점검 후 재 조정

#### 24) 자율주행 중 등간격이 일정하지 않음

* **원인:**
  1. 등간격 모드 OFF
  2. 작업기 유동 발생
* **조치 방법:**
  1. 156p 등간격  ON/OFF 항목을 본다.
  2. 작업기가 유동이 없도록 체크체인 고정

#### 25) 자율주행 중 등간격이 일정하게 변동됨

* **원인:**
  1. GNSS 수신기 횡방향 오차 발생
  2. 작업기 편차 발생
* **조치 방법:**
  1. GNSS 수신기 횡방향 오차 확인 후 보정 진행
  2. 바퀴자국으로 3방향 으로 주행 후 오차 발생 확인
  3. 작업기 수평 중앙 상태 확인
  4. 작업기로 3방향 으로 주행 후 오차발생 확인
  5. 177p  작업기 편차 계산 항목 확인 후 실행

#### 26) 자율주행이 갑자기 해제됨

* **원인:**
  1. RTK-GPS FIX 풀림
  2. 블루투스 연결 기기 전원 OFF
  3. 네트워크 끊어짐 발생
  4. 핸들 감도가 너무 약함.
* **조치 방법:**
  1. 228p 항목 확인 후 RTK-GPS 연결진행
  2. 블루투스 연결 기기 확인 후 조치
  3. USIM 데이터 및 테더링 데이터 확인
  4. 감도 조정 진행

#### 27) 태블릿 업데이트가 되지 않음

* **원인:**
  1. 네트워크 불량
* **조치 방법:**
  1. 네트워크 연결 확인
