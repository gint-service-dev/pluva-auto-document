# 고객 계정 준비

### 고객 계정 준비

퀵셋업에서 사용할 고객 계정을 사전에 준비합니다.\
해당 고객 계정 정보를 기반으로 맞춤 자율주행 서비스가 제공됩니다.



{% stepper %}
{% step %}
000페이지에 접속해 \[회원가입]을 누릅니다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=109-8671&t=SwZxMUaCLKxCsKdr-1" %}
{% endstep %}

{% step %}
이름, 휴대폰 번호, 비밀번호를 입력한 후 \[휴대폰 번호 인증하기]를 누릅니다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=109-8721&t=SwZxMUaCLKxCsKdr-1" %}

{% hint style="info" %}
이메일은 선택 항목으로, 입력하지 않아도 회원가입을 진행할 수 있습니다.
{% endhint %}
{% endstep %}

{% step %}
휴대폰 번호로 전송된 인증번호 6자리를 입력하고 \[인증완료]를 누릅니다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=3356-10287&t=SwZxMUaCLKxCsKdr-1" %}
{% endstep %}

{% step %}
휴대폰 번호 인증이 완료되면 약관 동의를 체크하고 \[가입완료]를 누릅니다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=3356-10299&t=SwZxMUaCLKxCsKdr-1" %}
{% endstep %}

{% step %}
회원가입이 완료됩니다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=3182-9475&t=SwZxMUaCLKxCsKdr-1" %}
{% endstep %}
{% endstepper %}
