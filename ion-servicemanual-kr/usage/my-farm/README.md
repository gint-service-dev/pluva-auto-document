# 내 농장 관리 (MY Farm)

### 내 농장 관리 (MY Farm)

My Farm은 농장과 필드 정보를 정리해 작업 환경을 관리하는 메뉴입니다.  필드, 작업 경로, 여유 구간(헤드랜드) 등을 한곳에서 관리할 수 있습니다.

{% hint style="info" %}
필드 등록, 헤드랜드 등록은 퀵셋업 작업과 동일하니 [퀵셋업-작업지 설정](../../order-installation/quick-setup/farm-setting.md)을 참고하세요.
{% endhint %}
