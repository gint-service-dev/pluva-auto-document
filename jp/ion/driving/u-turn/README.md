# 유턴

## 유턴

자율주행 농기계가 작업 라인의 끝에 도달했을 때, 자동으로 방향을 전환하여 다음 작업 라인으로 이동시키는 핵심 기능입니다. 작업 효율, 연료 절감, 작물 보호를 동시에 고려한 지능형 유턴 패턴을 제공합니다.

***

### 유턴 설정 방법

{% hint style="info" %}
유턴 타입 설정 → 2. 유턴 패턴 설정 → 3. 유턴 형태 설정
{% endhint %}

{% stepper %}
{% step %}
작업 버튼을 누르고 유턴 모드를 누릅니다.

<figure><img src="../../../.gitbook/assets/5-c-1.유턴 타입 설정(1) (2).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
유턴 항목을 누릅니다.

<figure><img src="../../../.gitbook/assets/5-c-1.유턴 타입 설정(2) (2).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
유턴 항목을 눌러 ON한다.

<figure><img src="../../../.gitbook/assets/5-c-1.유턴 타입 설정(3) (2).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
유턴 유형을 선택합니다.

<figure><img src="../../../.gitbook/assets/5-c-1.유턴 타입 설정(4) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
\[다음] 버튼을 누릅니다.

<figure><img src="../../../.gitbook/assets/5-c-1.유턴 타입 설정(5) (1).png" alt=""><figcaption></figcaption></figure>


{% endstep %}

{% step %}
사용할 유턴 패턴을 선택합니다.

<figure><img src="../../../.gitbook/assets/5-c-1.유턴 타입 설정(6) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
턴 방향과 세부 수치를 설정한 뒤 \[다음] 버튼을 누릅니다.

<figure><img src="../../../.gitbook/assets/5-c-1.유턴 타입 설정(7) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
유턴 형태를 선택합니다.

<figure><img src="../../../.gitbook/assets/5-c-1.유턴 타입 설정(8) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
\[확인] 버튼을 누릅니다.

<figure><img src="../../../.gitbook/assets/5-c-1.유턴 타입 설정(9) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
유턴 경로가 생성되며, 자율주행 버튼을 누르면 유턴이 포함된 자율주행이 시작됩니다.
