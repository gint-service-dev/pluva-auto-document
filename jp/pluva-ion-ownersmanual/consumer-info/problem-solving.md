# 문제해결 안내

### 문제해결안내

아래 목록에서 증상을 선택해 조치 방법을 확인합니다. 조치 후에도 문제가 지속되면 고객센터로 문의합니다.

***

#### 전원/연결

**Q: 전원이 들어오지 않습니다.**

* 전원 하네스 및 연결 상태를 확인해 주세요.
* 해결되지 않을 경우 구매처로 연락하여 조치를 받으세요.

**Q: GPS가 연결되지 않습니다.**

* 전원을 재부팅한 후, 개활지(하늘이 트인 곳)로 트랙터를 이동시켜 주세요.
* 해결되지 않을 경우 구매처로 연락하여 조치를 받으세요.



#### 앱/설정

**Q: OTA, S/W 업데이트가 진행되지 않습니다.**

* 전원 재부팅 후 GPS/LTE 수신이 원활한 개활지로 이동해 주세요.
* 해결되지 않을 경우 구매처로 연락하여 조치를 받으세요.

**Q: 앱 사용 중 오류가 발생합니다.**

* 태블릿 전원 버튼을 길게 눌러 시스템을 종료했다가, 다시 실행해 주세요.
* 해결되지 않을 경우 구매처로 연락하여 조치를 받으세요.



#### 자율주행

**Q: 자율주행 버튼에 불이 들어오지 않습니다.**

* 자율주행 준비 조건을 확인해 주세요.&#x20;
  * 속도: 0 km/h 이상 \~ 20 km/h 이하 - 진입 각도: 기준선에서 25도 이하
* 조건 충족 시에도 불이 안 켜지면 매니저에게 연락하세요.

**Q: 자율주행이 자동으로 해지됩니다.**

* 규정 속도를 준수해 주세요.
* 바퀴 각도가 너무 많이 틀어지면 해지됩니다



#### 작업 품질

**Q: 골이 일직선으로 똑바르게 나오지 않습니다.**

* 3단 히치에 작업기를 단단히 고정하여 유격이 없도록 해주세요.
* RPM을 올리거나 작업기를 살짝 올려 부하를 줄여주세요.

**Q: 등간격이 맞지 않습니다.**

* 링크 길이를 확인해 작업기를 정중앙에 맞춰주세요.
* 수평계를 이용해 작업기 수평을 맞춰주세요.
* 작업기 치수 설정값을 확인하고 재입력하세요.
* GNSS 리시버가 정중앙에 설치되었는지 확인하세요.



#### Q\&A

**Q: 어떤 기종(트랙터 / 이앙기)에 장착할 수 있나요?**

* 기본적으로 유압식 또는 기계식 스티어링을 가진 농용 트랙터와 이앙기 그리고 관리기 등 장착 가능합니다. 다만 차종·연식·스티어링 구조에 따라 호환 여부가 달라질 수 있으므로, 구체적인 호환 모델은 \[지원 모델 리스트] 또는 구매처에 문의해 주시기 바랍니다.

**Q: 장착은 사용자가 직접 할 수 있나요, 아니면 전문 기사 지원이 필요한가요?**

* 개인이 장착할 수는 있으나 기본적으로 공인 설치자 또는 교육을 받은 기술자에 의한 장착을 권장합니다. 전동 스티어링 휠, GNSS 수신기, IMU, 전원·CAN 배선 등이 안전과 성능에 직접 영향을 주기 때문입니다.

**Q: 플루바 아이온 자율주행 시스템을 시작하려면 어떤 조건이 필요하나요?**

* 기본적으로 다음 조건을 만족해야 합니다. 센서 신호 정상, RTKGPS 수신 상태, 작업 경로 설정 완료 가 기본적으로 설정되어야 합니다.

**Q: 플루바 아이온의 정확도는 어느 정도인가요?**

* RTK 보정신호 사용 시 일반적으로 센치미터급 수준입니다. 정확도는 기준국 품질, 안테나 설치, 주변 환경(가림, 반사)에 따라 달라질 수 있습니다.

**Q: 위험한 상황 시 플루바 아이온이 알아서 멈춰 주나요? (장애물 자동 인식 등)**

* 플루바 아이온은 주행 경로를 유지해 주는 보조 시스템이며, 앞을 보고 사람·장애물을 인지해 사고를 예방하는 ‘완전 자율주행 시스템’이 아닙니다. 항상 운전자가 전방·주변을 주시하면서 필요 시 즉시 브레이크 및 클러치 조작을 해야 합니다.
