# A+직진

### A+직진 설정

A점을 기준으로 설정한 각도의 대각선 직선 경로로 주행합니다.

{% stepper %}
{% step %}
\[A] 버튼을 눌러 A 지점을 생성합니다.

<figure><img src="../../.gitbook/assets/5-b-3.A+ 직진(1) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
\[A+] 버튼을 누르면 A+ 직진 모드 작업이 시작됩니다.

<figure><img src="../../.gitbook/assets/5-b-3.A+ 직진(2) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
경로 각도와 경로 길이 수치를 입력한 뒤 \[저장] 버튼을 누르면 경로가 저장됩니다.

<figure><img src="../../.gitbook/assets/5-b-3.A+ 직진(3) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
\[자율주행 버튼]을 눌러 자율 주행을 시작합니다.

<figure><img src="../../.gitbook/assets/5-b-3.A+ 직진(4) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
