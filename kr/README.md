{% hint style="info" %}
**참고**\
자율주행 키트 모델에 따라 제공되는 제품 매뉴얼의 종류 및 내용은 일부 다를 수 있습니다. 제품 매뉴얼의 내용 중 일부 장치 및 소프트웨어는 제조사 정책에 따라 변경될 수 있으며 버전이 다를 수 있습니다.
{% endhint %}

**주의**\

{% endhint %}
