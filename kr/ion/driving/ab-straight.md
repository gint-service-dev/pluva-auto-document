# AB직진

### AB직진 설정

A점과 B점을 잇는 방향으로 직진 주행합니다.

{% stepper %}
{% step %}
\[A] 버튼을 눌러 A 지점을 생성합니다.

<figure><img src="../../.gitbook/assets/5-b-2.AB 직진(1) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
10m 이상 직진 주행한 뒤, 원하는 지점에서 \[B] 버튼을 눌러 B 지점을 생성합니다.

<figure><img src="../../.gitbook/assets/5-b-2.AB 직진(2) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
AB 직진 경로가 생성되고 \[주행 시작] 버튼을 누르면 자율주행이 시작됩니다.

<figure><img src="../../.gitbook/assets/5-b-2.AB 직진(3) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
