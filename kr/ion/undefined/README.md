### 유턴 모드

유턴 모드는 자율주행 농기계가 작업 라인 끝에 도달했을 때 자동으로 방향을 전환하여 다음 작업 라인으로 이동하는 핵심 기능입니다. 작업 효율성, 연료 절약, 작물 보호를 동시에 고려한 지능형 유턴 패턴을 제공합니다.
