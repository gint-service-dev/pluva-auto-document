# 開通および取り付けのご案内

PLUVA iONは、専門的な取り付け工程が必要な製品です。**開通から取り付けまでのプロセス**を通じて、技術者による安全な取り付けをサポートします。

***

### 開通および取り付けプロセス

{% stepper %}
{% step %}
#### 製品の開通

* 取り付けする構成品を選択し、シリアル番号を登録して開通します。
* 詳細は、[<mark style="color:$primary;">製品の開通</mark>](product-registration.md)をご参照ください。
{% endstep %}

{% step %}
#### **取り付け**

* 車両に製品を取り付けます。既存のハンドルを自動操舵用のハンドルに交換し、各種センサーおよび機器を接続する作業が含まれます。
* 詳細は、[<mark style="color:$primary;">製品の取り付け</mark>](https://kuboda-pluva-servicemanual.pluva.io/ion/jp/order-installation/product-installation)をご参照ください。
{% endstep %}

{% step %}
#### **簡単セットアップ**

{% hint style="info" %}
簡単セットアップとは？

お客様が導入後すぐにタブレットの運用ができるよう、取り付けの際に**初期設定を行うステップ**です。
{% endhint %}

* ネットワークや車両情報など、システム作動に必要な**初期設定を完了**します。設定完了後には、製品がアクティベートされ利用可能な状態になります。
* 詳細は、[<mark style="color:$primary;">簡単セットアップ</mark>](https://kuboda-pluva-servicemanual.pluva.io/ion/jp/order-installation/quick-setup)をご参照ください。
{% endstep %}

{% step %}
#### **テスト走行**

* 取り付け完了後にはテスト走行を行います。正しく動作するかを確認し、**誤作動や異常の有無を確認**します。不具合が発生した際には、原因を調査し、適切な対応を実施します。
{% endstep %}

{% step %}
#### **お客様への操作説明**

* 製品の取り付けが完了されたら、安全な運用をサポートするための初期レクチャーを実施します。**主な機能や使用方法についてご案内**します。
* 詳細は、[<mark style="color:$primary;">使い方</mark>](https://kuboda-pluva-servicemanual.pluva.io/ion/jp/usage/customer-guidance)をご参照ください。
{% endstep %}

{% step %}
#### 取り付け完了

* 全ての取り付け及び設定が完了されたら、取り付け確認事項を点検し、取り付けを終了してください。
* 詳細は、[<mark style="color:$primary;">取り付け完了確認</mark>](installation-completed.md)をご参照ください。
{% endstep %}
{% endstepper %}
