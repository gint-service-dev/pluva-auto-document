# 取り付け前の準備

取り付けをスムーズに行うため、訪問前に必ず以下の内容をご確認ください。準備不足により部品の不適合や現場加工ができない場合、当日の取り付けができず後日の再訪問となる恐れがあります。

***

### 訪問前の準備段階

{% stepper %}
{% step %}
**日程及び訪問先の確認**

* **担当者の確認事項：**&#x53D6;り付ける製品のお客様と製品情報を予め確認します。
* **訪問先の正確な位置の事前確認：** 圃場は、住所だけでは特定が困難な場合が多いため、Googleマップのピン（位置情報）を共有頂くよう、 お客様にご案内してください。
{% endstep %}

{% step %}
**型番の互換性の確認および写真の確保**

農業機械の機種に適合する部品（スプライン、ブラケットなど）を事前に用意するため、以下の写真を訪問前に確保してください。

**必要な写真**

* **正面：**&#x30CF;ンドルの操作部全体の構成が分かる写真
* **側面：**&#x30CF;ンドルの軸の角度及びハンドル回りのスペースが確認できる写真

{% hint style="warning" %}
**取り付けできない機種**

タイヤ型及び半クローラーの農業機械は取り付けできますが、**フルクローラー機種は取り付けできません。**&#x8A2A;問前に型番を必ずご確認ください。
{% endhint %}
{% endstep %}

{% step %}
**GNSS受信機及びタブレットの取り付け位置のご相談**

取り付け位置は、測位精度や作業の利便性に影響を与えます。訪問前に、取り付け位置に関してお客様と予め相談してください。

* **キャビンのある農業機械：**&#x47;NSS受信機をキャビンの上に取り付けます。農業機械メーカー純正の受信機があるかどうかを予めご確認ください。
* **オープン型（屋根なし）：** 受信機を固定するための取り付け金具があるかをご確認ください。
* **タブレット：**&#x904B;転中に操作しやすく、しっかり固定できる位置に調整します。お客様と相談しながら確定してください。
{% endstep %}

{% step %}
**穴あけおよび改造の際の事前同意**

取り付け中に、やむを得ず機体を加工する場合があります。訪問前にお客様へ以下の内容をご案内した上で同意を得てください。

* **クラクション：** スイッチの取り付け場所の変更、配線作業時に機体への穴あけが必要となる可能性があります。
* **田植え機のハンドル軸カバー：** Uボルトブラケットを取り付けるために、カバーへの穴開けが必要となる可能性があります。
* **田植え機のGNSS受信機のステー：** 田植え機に取り付ける際には、受信機専用のステーが必ず必要です。お客様の方で事前準備が必要であり、ステーがない場合は、取り付け当日に受信機の取り付けができません。
{% endstep %}

{% step %}
**通信環境およびアカウントの準備**

* **インターネット接続：**&#x7C21;単セットアップやRTK測位に向けてセルラー（LTE）へ接続する必要があります。Simカードを用意し、またはスマートフォンからのテザリングが可能かどうかを予めご確認ください。
* **接続予定のRTKの確認：**&#x7C21;単セットアップから位置補正方法を設定します。訪問前にお客様がどのRTKに接続予定なのかを確認し、必要な情報を予め確保します。
  * **RTK手動接続：**&#x4E;TRIP方式の補正信号を受信します。訪問前にお客様の方で以下の情報を予め準備していただくよう案内してください。　
    * 基地局、サーバー住所およびポート
    * マウントポイント
    * アカウントIDおよびパスワード
  * **RTK Bluetooth接続：**&#x5916;部のRTK受信機とBluetooth接続します。訪問前にお客様の方でRTKアプリのアカウントIDとパスワードをご用意するよう案内してください。
* **PLUVA会員登録：**&#x50;luva iONを使用するためには、お客様のアカウントが必要です。取り付け前に会員登録を完了したかどうかご確認ください。[お客様アカウントの準備](preparing-accounts.md)をご参照ください。

{% hint style="warning" %}
ソフトウェアアップデート（OTA）を行うと、約**300MB～500MB**のデータ通信量が発生します。お客様のデータ料金プランを予め確認のうえ、データが足りない場合は、Wi-Fiやテザリングを用意してください。
{% endhint %}
{% endstep %}

{% step %}
**試運転及びオートキャリブレーションのスペース確保**

取り付け後には自動補正（オートキャリブレーション）のための走行が必要となります。以下の条件を満たすスペースを確保していただくよう、訪問前にお客様へ依頼してください。

* **地面の状態：**&#x30BF;イヤが滑らない、平らで硬い地面（コンクリート、または平坦な土壌）
* **必要な面積：**
  * 通常の機種：最低**30m × 30m**以上
  * 倍速ターン付き機種：最低**50m × 40m**以上

{% hint style="warning" %}
以上の準備が整っていない場合、部品の不適合や現場での加工が困難となり、取り付け日程が延期される恐れがあります。準備の際にご不明な点がございましたら(株)GINTまでお問い合わせください。
{% endhint %}
{% endstep %}
{% endstepper %}

***

### 訪問前の最終チェックリスト

お客様の訪問先へ出発する前に、下記のチェックリストを順にご確認ください

<details>

<summary>事前確認</summary>

* [ ] 担当者確認：講習修了証の有無を確認する
* [ ] 訪問先の位置：訪問先の住所およびGoogleマップのピンを事前確認
* [ ] 型番の互換性について最終確認：
  * トラクターor田植え機の正確な型番を確認する（フルクローラー機種は取り付け不可）
  * ハンドル周辺の写真を確認する
  * キャビンの有無を確認する（受信機取り付け用の金具の有無を確認）
  * 構成品の取り付け位置を確認する
  * クラクションを取り付けるため、お客様へ機体に穴あけが必要であることをご案内をし、事前に同意を得る
  * 田植え機への取り付けの際：GNSS受信機のステーが用意されているかを確認する（ステーがない場合、受信機の取り付け不可）
  * 田植え機への取り付けの際：Uボルトブラケットを装着するため、お客様へハンドル軸カバーに穴あけが必要であることをご案内し、事前に同意を得る
* [ ] PLUVAアカウント：ユーザーのPLUVA会員登録が完了したかを確認する
* [ ] お客様がご使用予定のRTKサービスの接続情報を確認する
* [ ] オートキャリブレーションのためのスペースを確保する：自動補正（キャリブレーション）のための平らな作業スペース（30m × 30m以上）をご用意いただくよう案内する

</details>
