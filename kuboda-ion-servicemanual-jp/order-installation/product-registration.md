# 製品の開通

製品を登録し、開通キーを発行します。取り付けを円滑に進めるため、取り付け前に開通することを推奨します。

***

#### 注文製品ごとの開通に必要な構成品

注文いただいた製品ごとに、下記の構成品をご用意ください。



1. **PLUVA iON**

* 全ての主な構成品を登録します。
  * タブレット
  * GNSS受信機
  * 電動ステアリングホイール



2. **エクスパンションキット（拡張キット）**

* タブレット以外の構成品を登録します。
  * GNSS受信機
  * 電動ステアリングホイール



3. **追加オプション**

* ワンタッチスイッチ



***

#### シリアル番号の登録（梱包番号）

製品の登録は、製品に付着してあるQRコード（シリアル番号、または梱包番号）をスキャンして進めます。

* 梱包番号（外箱のQRコード）を登録すると、構成品を**一括登録**できます。

#### 製品の開通方法

{% stepper %}
{% step %}
製品の開通ページにアクセスし、\[QRをスキャンし社員番号を入力]をタップします。

<figure><img src="../.gitbook/assets/product-opening-1.png" alt="" width="338"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
社員番号のQRコードをスキャンします。

<figure><img src="../.gitbook/assets/product-opening-2.png" alt="" width="270"><figcaption></figcaption></figure>

{% hint style="info" %}
QRスキャンできない場合は、\[社員番号の直接入力]をタップし、社員番号を手動で入力します。
{% endhint %}


{% endstep %}

{% step %}
「開通する製品の選択」ページにて注文製品および追加オプション情報を設定してから\[次へ]をタップします。

<figure><img src="../.gitbook/assets/product-opening-3.png" alt="" width="334"><figcaption></figcaption></figure>


{% endstep %}

{% step %}
「主な製品の開通」ページにて\[パッケージでまとめて開通する]をタップします。

<figure><img src="../.gitbook/assets/product-opening-4.png" alt="" width="334"><figcaption></figcaption></figure>

{% hint style="info" %}
各構成品をそれぞれ登録することもできます。
{% endhint %}
{% endstep %}

{% step %}
梱包番号のQRコードをスキャンします。

<figure><img src="../.gitbook/assets/product-opening-5.png" alt="" width="270"><figcaption></figcaption></figure>

{% hint style="info" %}
カメラスキャンによるコード入力ができない場合は、入力欄をタップし、直接入力してください。

<img src="../.gitbook/assets/directly-enter.png" alt="" data-size="original">
{% endhint %}
{% endstep %}

{% step %}
梱包番号を確認し、\[確認完了]をタップします。

<figure><img src="../.gitbook/assets/product-opening-6.png" alt="" width="311"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
登録完了されると、主な製品の開通ポップアップから\[開通完了]をタップします。

<figure><img src="../.gitbook/assets/product-opening-7.png" alt="" width="329"><figcaption></figcaption></figure>

{% hint style="info" %}
梱包番号（シリアル番号）が無効な場合、QRコードのスキャン画面に戻ります。

<img src="../.gitbook/assets/product-opening-invaild.png" alt="" data-size="original">
{% endhint %}

{% hint style="info" %}
追加オプション品をご注文の場合は、主要製品の開通後に、追加オプションの開通を完了させることで、全体の開通が完了となります。

<img src="../.gitbook/assets/opening-additional-options.png" alt="" data-size="original">
{% endhint %}

{% hint style="info" %}
再度、製品を開通および選択する場合は、\[製品を再選択]をタップし、製品の選択ページへ戻ります。

<img src="../.gitbook/assets/product-reset.png" alt="" data-size="original">
{% endhint %}
{% endstep %}

{% step %}
製品の開通が完了されます。

<figure><img src="../.gitbook/assets/product-opening-8.png" alt="" width="270"><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
