# 설치 완료 확인

설치 및 퀵셋업 작업의 완료 항목을 점검하고 최종 설치 상태를 확인하는 단계입니다.

{% hint style="info" %}
**모든 설치 작업을 완료한 후 아래 항목을 확인합니다.**
{% endhint %}

{% stepper %}
{% step %}
제품이 정상적으로 장착되었습니다.
{% endstep %}

{% step %}
필요한 소프트웨어 설정이 모두 완료되었습니다.
{% endstep %}

{% step %}
고객 교육이 완료되었으며, 고객이 즉시 서비스를 이용할 수 있는 상태입니다.
{% endstep %}

{% step %}
시범 운행을 통해 자율주행 가능 여부를 확인합니다.
{% endstep %}
{% endstepper %}
