# 取り付けチケットでの製品開通

取り付けチケットに製品を登録し、開通キーを発行します。取り付けを円滑に進めるため、取り付け前に開通することを推奨します。

{% hint style="info" %}
取り付けチケットとは？

注文製品ごとの**取り付け状況を確認できる管理チケット**です。
{% endhint %}

***

#### 注文製品ごとの開通に必要な構成品

注文いただいた製品ごとに、下記の構成品を予めご用意ください。

1. **pluva ion**

* 全ての主な構成品を登録します。
  * タブレット
  * GNSS受信機
  * 電動ステアリングホイール

2. **pluva ion エクスパンションキット（拡張キット）**

* タブレット以外の構成品を登録します。
  * GNSS受信機
  * 電動ステアリングホイール

3. 追加オプション

* ワンタッチスイッチ

***

#### シリアル番号の登録（梱包番号）

製品の登録は、製品に付着してあるQRコード（シリアル番号、または梱包番号）をスキャンして進めます。

* 梱包番号（外箱のQRコード）を登録すると、構成品を**一括登録**できます。

#### 取り付けチケットへのアクセス方法

{% stepper %}
{% step %}
[アドミンページ](https://gint-admin.pluva.jp/)にログインします。

<figure><img src="../.gitbook/assets/enter-ticket-1.png" alt="" width="348"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
取り付けチケット一覧\
​\
取り付けチケット一覧にアクセスし、ご希望のチケットを選択します。

<figure><img src="../.gitbook/assets/enter-ticket-2.png" alt="" width="336"><figcaption></figcaption></figure>

{% hint style="info" %}
左上のメニュー（三）をクリックし「注文及び取り付け管理」 > 「取り付けチケット一覧」へアクセスできます。

<img src="../.gitbook/assets/enter-ticket-list.png" alt="" data-size="original">
{% endhint %}
{% endstep %}
{% endstepper %}

***

#### 製品の開通方法

{% stepper %}
{% step %}
取り付けチケットから\[製品の開通開始]を選択します。

<figure><img src="../.gitbook/assets/enter-ticketdetail-1.png" alt="" width="323"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
\[パッケージでまとめて開通する]をタップします。

<figure><img src="../.gitbook/assets/enter-ticketdetail-2.png" alt="" width="320"><figcaption></figcaption></figure>

{% hint style="info" %}
各構成品を選択すると、個別での登録もできます。
{% endhint %}
{% endstep %}

{% step %}
梱包番号のQRコードをスキャンします。

<figure><img src="../.gitbook/assets/enter-ticketdetail-3.png" alt="" width="270"><figcaption></figcaption></figure>

{% hint style="info" %}
カメラのスキャンで正しくコードが入力されない場合は、入力欄をタップし、直接入力してください。

<img src="../.gitbook/assets/number-direct-input.png" alt="" data-size="original">
{% endhint %}
{% endstep %}

{% step %}
梱包番号を確認し\[確認完了]をタップします。

<figure><img src="../.gitbook/assets/enter-ticketdetail-4.png" alt="" width="315"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
登録が完了したら、「主な製品の開通」ポップアップから\[開通完了]をタップします。

<figure><img src="../.gitbook/assets/enter-ticketdetail-5.png" alt="" width="311"><figcaption></figcaption></figure>

{% hint style="info" %}
無効な梱包番号（シリアル番号）の場合、QRスキャン画面に戻ります。

<img src="../.gitbook/assets/invalid-ticket.png" alt="" data-size="original">
{% endhint %}

{% hint style="info" %}
追加オプション品をご注文の場合は、主な製品の開通後に追加オプションの開通を進めることで、開通が完了します。

<img src="../.gitbook/assets/addoption.png" alt="" data-size="original">
{% endhint %}
{% endstep %}

{% step %}
製品の開通が完了します。

<figure><img src="../.gitbook/assets/enter-ticketdetail-6.png" alt="" width="270"><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
