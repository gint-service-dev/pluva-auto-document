# 9. (앱) 작업 중 관성센서 보정

## 5-9. 작업 중 관성센서 교정

{% hint style="info" %}
차량이 직진하지 않고 흔들리거나 휘어질 때 내장된 관성센서(IMU)를 자동으로 교정해 정확한 직진 주행을 돕는다. 센서 교정은 약 1분 내외로 완료된다.
{% endhint %}
