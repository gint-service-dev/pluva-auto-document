# 7. 주행 수치 보정

## 7-7. 주행 수치 보정 <a href="#id-7-5" id="id-7-5"></a>

플루바 앱에서 설정했던 전문적인 주행 수치 보정을 동일하게 조정할 수 있는 기능이다.

{% hint style="danger" %}
주행 수치 보정은 임의 변경 시 자율주행 성능에 영향을 줄 수 있으므로 신중히 변경하며, 변경 시 반드시 고객과 실시간으로 소통하며 진행한다.
{% endhint %}
