# 설치 전 대리점 준비 가이드

현장 방문 전 아래 항목을 반드시 확인하고 준비합니다. 사전 준비가 미흡하면 부품 불일치나 현장 가공 불가로 인해 설치 일정이 지연될 수 있습니다.

***

### 일정 및 현장 정보 확인

* **담당자 확인**: 설치 담당자의 성함, 연락처, 교육 수료증 소지 여부를 확인합니다.
* **위치 공유**: 농경지는 주소만으로 위치 특정이 어려운 경우가 많습니다. 구글맵(Google Maps) 위치 핀을 반드시 공유받습니다.

***

### 기종 호환성 확인 및 사진 자료 확보

적합한 부품(스플라인, 브래킷 등)을 사전에 준비하기 위해 아래 사진 자료를 방문 전에 확보합니다.

#### 필요 사진

* **정면**: 핸들 조작부 전체 구성 확인
* **측면**: 핸들 축 각도 및 주변 공간 확인

#### 기종 설치 가능 여부

| 기종 유형 | 설치 가능 여부 |
|---|---|
| 타이어형 | 가능 |
| 하프 크롤러 | 가능 |
| 풀 크롤러(Full Crawler) | 불가 |

{% hint style="info" %}
모델명을 정확히 확인합니다. (예: 쿠보타 MR1000)
{% endhint %}

***

### GNSS 수신기 및 태블릿 설치 위치 협의

설치 위치에 따라 측위 정밀도와 작업 편의성이 달라지므로, 방문 전 농가와 사전 협의합니다.

* **캐빈(Cabin) 모델**: 캐빈 위에 장착합니다. 제조사 순정 GNSS 수신기 유무를 미리 확인합니다.
* **오픈형(지붕 없음)**: 수신기를 고정할 별도 스테이(거치대) 유무를 확인합니다.
* **이앙기**: GNSS 수신기 전용 거치대가 반드시 필요합니다. 고객이 방문 전 준비해야 하며, 미준비 시 수신기 설치가 불가합니다.
* **태블릿**: 운전 중 조작이 편리하고 견고하게 고정할 수 있는 위치를 농가와 협의합니다.

***

### 타공 및 개조 사전 동의 확보

설치 과정에서 기체 일부에 가공이 필요할 수 있습니다. 방문 전 고객에게 아래 내용을 안내하고 동의를 받습니다.

* **클락션(경적)**: 스위치 이식 또는 배선을 위한 타공이 발생할 수 있습니다.
* **이앙기 핸들 커버**: U볼트 브래킷 장착을 위해 타공이 필요할 수 있습니다.

***

### 통신 환경 및 계정 준비

* **인터넷 연결**: 퀵셋업 및 RTK 측위를 위해 LTE 연결이 필수입니다. USIM 카드를 준비하거나 휴대폰 테더링 가능 여부를 사전에 확인합니다.
* **RTK 서비스 계정**: 사용 예정인 RTK 서비스의 접속 정보를 사전에 확보합니다.
* **PLUVA 회원가입**: 퀵셋업의 차량 등록 단계에서 고객 계정이 필요합니다. 방문 전 가입 완료 여부를 확인합니다. ([고객 계정 준비](preparing-accounts.md) 참고)

{% hint style="warning" %}
소프트웨어 업데이트(OTA) 진행 시 약 **300MB \~ 500MB**의 데이터가 사용됩니다. 고객의 데이터 요금제를 사전에 확인하고, 데이터가 충분하지 않은 경우 Wi-Fi 또는 테더링 환경을 준비합니다.
{% endhint %}

***

### 오토 캘리브레이션 공간 확보

제품 설치 후 정밀한 자동 보정(오토 캘리브레이션) 주행이 필요합니다. 아래 조건의 공간을 고객에게 미리 요청합니다.

* **지면 상태**: 바퀴가 미끄러지지 않는 평평하고 단단한 지면 (콘크리트 또는 잘 다져진 흙)
* **필요 면적**:
  * 일반 기종: 최소 **30m × 30m** 이상
  * 배속턴 기종: 최소 **50m × 40m** 이상

***

### 방문 전 최종 체크리스트

#### 사전 확인

* [ ] 담당자 교육 수료증 소지 여부 확인
* [ ] 구글맵 위치 핀 공유 완료
* [ ] 기종 모델명 확인 및 풀 크롤러 여부 검토
* [ ] 핸들 주변 사진 확보 (정면·측면)
* [ ] 캐빈 유무 및 GNSS 거치대 준비 상황 확인
* [ ] 타공 관련 고객 사전 동의 완료
* [ ] PLUVA 회원가입 완료 여부 확인
* [ ] RTK 서비스 계정 보유 여부 확인
* [ ] 오토 캘리브레이션 공간 확보 요청 완료
* [ ] 고객 데이터 요금제 확인 (OTA 업데이트용)

#### 지참 물품

* [ ] 교육 수료증
* [ ] 제품 (PLUVA iON KIT / EXPANSION KIT)
* [ ] 스플라인 / 브래킷 / U볼트
* [ ] 원터치 스위치
* [ ] USIM 카드
* [ ] 타공 도구

***

준비가 완료되면 [제품 설치](product-installation/)를 진행합니다.
