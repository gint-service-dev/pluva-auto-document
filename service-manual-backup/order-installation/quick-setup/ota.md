# 소프트웨어 업데이트(OTA)

#### 소프트웨어 업데이트(OTA) **이란?**&#x20;

최신 성능으로 안정적인 사용 환경을 제공하기 위해 기기가 자동으로 최신 버전을 확인하고 업데이트합니다.

{% hint style="info" %}
최신 업데이트가 필요한 경우에만 소프트웨어 설정 화면이 표시됩니다.
{% endhint %}

{% hint style="warning" %}
소프트웨어 업데이트(OTA) 진행 시 약 **300MB \~ 500MB**의 데이터가 사용됩니다. 고객의 데이터 요금제를 사전에 확인하고, 데이터가 충분하지 않은 경우 Wi-Fi 또는 테더링 환경을 준비합니다.
{% endhint %}

***

{% stepper %}
{% step %}
\[업데이트] 버튼을 누릅니다.

<figure><img src="../../.gitbook/assets/Quick setup_OTA-1.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
와이파이 환경에서 소프트웨어 업데이트를 진행할 것을 권장합니다.
{% endhint %}

{% hint style="info" %}
네트워크가 셀룰러로 설정되어 있는 경우 \[네트워크 변경]을 통해 변경할 수 있습니다.
{% endhint %}
{% endstep %}

{% step %}
업데이트가 진행됩니다.

<figure><img src="../../.gitbook/assets/Quick setup_OTA-2.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
업데이트가 완료되면 \[재부팅] 버튼을 누릅니다.

<figure><img src="../../.gitbook/assets/Quick setup_OTA-3.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
재부팅 후 태블릿에 다시 접속하여 \[다음 단계로] 버튼을 누르면 소프트웨어 업데이트가 완료됩니다.

<figure><img src="../../.gitbook/assets/Quick setup_OTA-4.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
