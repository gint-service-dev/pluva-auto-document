# 주행모드(경로플래닝)

### 주행모드 (경로플래닝)

주행 모드는 사용자가 설정한 경로를 따라 농기계가 자동으로 이동하며 작업하도록 돕는 기능입니다. 주행 중에는 주행 모드, 속도, 작업 면적/시간, 차량/작업기 상태가 화면에 표시됩니다. 이를 통해 현재 작업 상황을 한눈에 확인하고, 필요할 때 자율주행과 수동 운전을 안전하게 전환할 수 있습니다.
