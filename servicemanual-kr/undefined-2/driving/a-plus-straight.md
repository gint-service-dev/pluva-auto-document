# 설치 티켓 조회

### 설치 티켓 조회

등록된 주문서를 조회하는 화면입니다. 주문서는 최신 생성순으로 나열되며, 검색을 통해 원하는 주문을 찾을 수 있습니다.

***

#### 설치 티켓 목록

{% stepper %}
{% step %}
\[주문/설치 관리] > \[설치 티켓 목록] 메뉴에서 티켓 목록을 확인할 수 있다. 티켓을 클릭하면 상세 정보를 확인할 수 있습니다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=1494-9896&t=LwWj6XmqVSqNcRa9-1" %}
{% endstep %}
{% endstepper %}

![](<../../.gitbook/assets/icon-square-1 (1).svg>) **검색 창**

* 설치티켓 번호 또는 장소명을 입력해 원하는 주문 목록을 검색합니다.

![](../../.gitbook/assets/icon-square-2.svg) **설치 품목**

* 설치 품목을 표시합니다.

![](../../.gitbook/assets/icon-square-3.svg) **장소명**

* 주문 작성 시 입력한 장소명을 표시합니다.

![](../../.gitbook/assets/icon-square-4.svg) **담당 조직 / 담당자**

* 담당 조직을 표기합니다.
* 설치티켓 담당자를 표기합니다. 담당자 미정시 표시되지않습니다.

![](<../../.gitbook/assets/icon-square-5 (1).svg>) **상태**

* 설치 티켓 상태를 표시합니다.
  * 발급: 설치 작업을 위해 티켓이 생성된 상태로, 취소 처리가 가능합니다.
  * 설치중: 현장에서 설치티켓의 제품등록이 완료된 상태입니다.
  * 완료:  퀵셋업이 완료되고 설치티켓의 설치 완료 확인을 끝낸 상태입니다.
  * 취소: 설치 작업이 중단되거나 취소된 상태로, 해당 티켓으로는 더 이상 설치 진행이 불가능합니다.

![](../../.gitbook/assets/icon-square-6.svg) **설치 티켓 번호**

* 설치 티켓의 고유 번호를 표시합니다.
  * 번호 규칙: 설치티켓 생성일시 - 구분자,대리점코드 - 랜덤코드 입니다.

***

#### 설치 티켓 상세 페이지

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=1535-11648&t=LwWj6XmqVSqNcRa9-1" %}

![](<../../.gitbook/assets/icon-square-1 (1).svg>) **설치 티켓 번호**

* 설치 티켓의 고유 번호를 표시합니다.

![](../../.gitbook/assets/icon-square-2.svg) **상태**

* 설치 티켓 상태를 표시합니다.

![](../../.gitbook/assets/icon-square-3.svg) **티켓 발급일**

* 티켓 발급일을 표시합니다.

![](../../.gitbook/assets/icon-square-4.svg) **담당자**

* 담당자를 표기합니다. 담당자 미정시 담당자 배정 버튼이 노출됩니다.

![](../../.gitbook/assets/icon-square-5.svg) **주문 제품 정보**

* 주문 제품 정보를 표기합니다.
* 티켓 상태에 따라 아래 버튼이 노출됩니다.
  * 발급: 제품 등록 시작&#x20;
  * 설치중: 설치 확인 완료
  * 완료: 버튼 없음

![](../../.gitbook/assets/icon-square-6.svg) **차량 정보**

* 주문 등록 시 입력했던 차량 정보가 노출됩니다.

![](../../.gitbook/assets/icon-square-7.svg) **방문 정보**

* 주문 등록 시 입력했던 방문 장소 정보가 노출됩니다.
