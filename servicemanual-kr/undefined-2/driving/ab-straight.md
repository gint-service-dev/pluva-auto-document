# 주문 취소

### 주문 취소

등록한 주문을 취소합니다.

{% hint style="danger" %}
유효 상태의 주문서만 취소할 수 있다. 취소한 주문은 되돌릴 수 없으므로 충분히 확인 후 진행해야 합니다.
{% endhint %}

{% hint style="danger" %}
설치 중인 티켓이 있다면 주문을 취소할 수 없습니다.
{% endhint %}

***

#### 주문 취소 방법

{% stepper %}
{% step %}
\[주문 관리] > \[주문 목록 관리] 메뉴에서 주문서를 클릭하여 상세 화면에 접속한다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=1494-5860&t=LwWj6XmqVSqNcRa9-1" %}
{% endstep %}

{% step %}
\[주문 취소] 버튼을 클릭한다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=1494-7109&t=LwWj6XmqVSqNcRa9-1" %}
{% endstep %}

{% step %}
주문 취소 팝업에서 \[주문 취소]를 누릅니다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=1494-8885&t=LwWj6XmqVSqNcRa9-1" %}
{% endstep %}

{% step %}
주문 취소가 완료됩니다.

{% endstep %}
{% endstepper %}
