# 주문서 조회

### 주문서 조회

등록된 주문서를 조회하는 화면입니다. 주문서는 최신 생성순으로 나열되며, 검색을 통해 원하는 주문을 찾을 수 있습니다.

{% stepper %}
{% step %}
\[주문/설치 관리] > \[주문 목록] 메뉴에서 주문 목록을 확인할 수 있다. 항목을 클릭하면 상세 정보를 확인할 수 있습니다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=1457-5398&t=LwWj6XmqVSqNcRa9-1" %}
{% endstep %}
{% endstepper %}

![](<../../.gitbook/assets/icon-square-1 (1).svg>) **검색 창**

* 주문번호 또는 장소명을 입력해 원하는 주문 목록을 검색합니다.

![](<../../.gitbook/assets/icon-square-2 (1).svg>) **주문번호**

* 각 주문의 고유 번호를 표시합니다.
  * 번호 규칙: 주문서 생성일시 - 구분자/대리점코드 - 랜덤코드 입니다.

![](<../../.gitbook/assets/icon-square-3 (1).svg>) **상태**

* 주문서 상태를 표시합니다.
  * 유효: 고객이 정상적으로 주문을 완료하여 시스템에 등록된 상태로, 수정 및 취소가 가능합니다.
  * 완료: 주문부터 설치까지 모든 프로세스가 성공적으로 완료된 상태로, 수정 및 취소가 가능합니다.
  * 취소: 주문 프로세스가 중단된 상태로, 추가 처리나 수정이 불가능합니다.

![](<../../.gitbook/assets/icon-square-4 (1).svg>) **장소명**

* 주문 작성 시 입력한 장소명을 표시합니다.

![](<../../.gitbook/assets/icon-square-5 (1).svg>) **담당 조직**

* 담당 조직을 표기합니다.

![](<../../.gitbook/assets/icon-square-6 (1).svg>) **담당자**

* 주문 담당자를 표시합니다. 작성자가 자동으로 표시됩니다.

![](<../../.gitbook/assets/icon-square-6 (1).svg>) **주문일**

* 주문 등록일이 표기됩니다.
