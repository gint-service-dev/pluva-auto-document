# 작업기 추가

### 작업기 추가 (선택)

작업에 사용할 작업기를 추가합니다.

***

#### 작업기 추가 방법

{% stepper %}
{% step %}
\[추가하기]버튼을 누릅니다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=425-13924&t=PM760HAi58K4ZabD-1" %}
{% endstep %}

{% step %}
별칭, 작업기 타입, 작업기 너비, 고랑폭, 링크-작업기 거리, 작업기 편차를 입력하고 \[확인]을 누릅니다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=425-14118&t=PM760HAi58K4ZabD-1" %}
{% endstep %}

{% step %}
작업기 등록이 완료됩니다. \[다음 단계로] 버튼을 누르면 차량 보정에 진입합니다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=425-14119&t=PM760HAi58K4ZabD-1" %}
{% endstep %}
{% endstepper %}
