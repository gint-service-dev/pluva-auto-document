# 주문 등록

### 주문 등록&#x20;

주문의 기본 정보를 작성합니다. 주문 제품, 설치할 차량, 방문 정보를 입력하면 설치티켓이 자동 생성되며, 설치에 필요한 사항들을 미리 확인하고 준비할 수 있습니다.



{% stepper %}
{% step %}
주문 목록에서 \[주문 작성]을 누릅니다.

{% endstep %}

{% step %}
담당 조직을 선택합니다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=27-5470&t=PM760HAi58K4ZabD-1" %}

{% hint style="info" %}
담당 조직은 내 소속 조직이 기본으로 선택됩니다.
{% endhint %}
{% endstep %}

{% step %}
\[제품 추가]를 누릅니다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=16-3671&t=PM760HAi58K4ZabD-1" %}
{% endstep %}

{% step %}
주문할 제품을 선택한 뒤 \[추가]를 누릅니다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=21-4450&t=PM760HAi58K4ZabD-1" %}

{% hint style="info" %}
플루바 아이온과 Expansion Kit (확장키트)

* 플루바 아이온
  * GNSS 수신기, 전동 스티어링 휠, 태블릿이 포함된 기본 구성입니다.
  * 신규 고객이거나, 여러 농기계를 동시에 운영해 각 장비별로 태블릿이 필요한 고객에게 적합한 세트입니다.
  *

      <figure><img src="../.gitbook/assets/full-set kr.png" alt=""><figcaption></figcaption></figure>
* Expansion Kit (확장키트)
  * GNSS 수신기, 전동 스티어링 휠, 스위치가 포함됩니다. 태블릿은 포함되지 않습니다.
  * 한 대의 태블릿을 여러 농기계에서 활용하려는 고객에게 적합한 세트입니다.&#x20;
  * 확장키트는 태블릿 없이 단독으로 사용할 수 없으므로, 주문 시 고객이 이미 태블릿을 보유 중인지 또는 태블릿 포함 세트와 함께 구매하는지 확인이 필요합니다.
  *

      <figure><img src="../.gitbook/assets/Ion-extension-kr.png" alt=""><figcaption></figcaption></figure>
{% endhint %}

{% hint style="info" %}
추가 옵션

* 추가 옵션은 세트와 함께 구매할 수 있는 항목으로, 필요한 수량만큼 주문에 추가할 수 있습니다.
* 스위치는 세트에 포함되지 않으므로 추가 옵션으로 선택해 주세요.
{% endhint %}
{% endstep %}

{% step %}
주문 제품 정보를 입력합니다.

{% embed url="https://www.figma.com/design/3caVmkxMdvjppUG5NWSjbb/%EC%B0%A8%EC%84%B8%EB%8C%80-%EB%8C%80%EB%A6%AC%EC%A0%90-%EB%A9%94%EB%89%B4%EC%96%BC?node-id=27-6446&t=PM760HAi58K4ZabD-1" %}

{% hint style="info" %}
여러 세트를 주문하는 경우 \[제품 추가]를 눌러 항목을 추가합니다.

* 예1. 트랙터 2대, 이앙기 1대에 모두 플루바 아이온을 설치하고 싶은 신규 고객
  * \[제품 추가] 버튼을 눌러 총 3개의 플루바 아이온을 주문합니다.
* 예2. 플루바 아이온을 트랙터에 작년 설치했는데 이번에 이앙기를 새로 구매하여 태블릿을 옮겨서 사용하려는 고객
  * \[제품 추가] 버튼을 눌러 총 1개의 Expansion Kit를 주문합니다.
{% endhint %}

{% hint style="info" %}
제조사/모델 목록에 원하는 항목이 없으면 체크박스를 선택해 직접 입력합니다.
{% endhint %}
