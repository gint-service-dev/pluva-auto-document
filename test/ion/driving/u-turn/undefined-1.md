# 유턴 패턴 설명

### 유턴 패턴 설명

<figure><img src="../../../.gitbook/assets/5-c-2.유턴 타입 화면 설명(2).png" alt=""><figcaption></figcaption></figure>

![](../../../.gitbook/assets/1.svg) **유턴패턴**

* 즉시 턴: 생성된 모든 골의 유턴을 즉시 이동하는 회전 방식입니다.
* 골띄기 턴: 작업 라인 끝에서 다음 줄을 자동으로 건너 이동하는 회전 방식입니다.
* 단방향 턴: 생성된 모든 골을 한쪽 방향으로만 이동하는 회전 방식입니다.
* 확장 턴: 정사각형에 가까운 완만한 곡선으로 이동하는 회전 방식입니다.

![](../../../.gitbook/assets/2.svg) **유턴 패턴 미리보기**

* 현재 설정된 유턴 패턴이 실제 작업 라인에 어떻게 적용되는지 미리 확인할 수 있습니다.

![](../../../.gitbook/assets/3.svg) **경계선 오프셋**

* 작업 경계선으로부터 얼마나 떨어진 지점에서 유턴할지 설정합니다.

![](../../../.gitbook/assets/4.svg) **행 건너뛰기**

* 유턴 시 건너뛸 행(골)의 개수를 설정합니다.

![](../../../.gitbook/assets/4.svg) **최소 회전 반경**

* 유턴할 때의 최소 회전 반경을 설정합니다.

![](../../../.gitbook/assets/4.svg) **시작 방향**

* 직진 주행 후 첫 번째로 회전할 방향(좌회전/우회전)을 설정합니다.
