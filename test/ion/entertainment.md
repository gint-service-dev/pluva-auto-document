# 엔터테인먼트

엔터테인먼트에서는 YouTube를 시청할 수 있습니다. 주행 화면 우측에 YouTube 플레이어가 표시됩니다.

{% hint style="info" %}
**엔터테인먼트 기능은 자율주행 중 편의를 위해 제공되지만, 차량은 항상 운전자가 주시해야 합니다.**

* 자율주행 중에도 전방 및 주변 상황(장애물, 사람, 동물 등)을 항상 확인하세요.
* 미디어 조작은 차량이 정지한 상태이거나 안전한 상황에서 수행하세요.
* 볼륨이 너무 크면 외부 경고음이나 주변 소음을 듣지 못할 수 있으니 적정 볼륨을 유지하세요.
{% endhint %}
