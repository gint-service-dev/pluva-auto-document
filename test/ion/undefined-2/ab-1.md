# AB커브

### AB커브 설정

A점과 B점을 잇는 곡선을 따라 커브 주행합니다.

{% stepper %}
{% step %}
\[A] 버튼을 눌러 A 지점을 생성합니다.

<figure><img src="../../.gitbook/assets/5-b-4.AB 커브(1) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
커브 작업을 진행할 경로를 따라 주행한 뒤, 원하는 지점에서 \[B] 버튼을 눌러 B 지점을 선택합니다.

<figure><img src="../../.gitbook/assets/5-b-4.AB 커브(2) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
AB 커브 경로가 생성된다. \[자율주행 시작] 버튼을 누르면 자율주행이 시작됩니다.

<figure><img src="../../.gitbook/assets/5-b-4.AB 커브(3) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
