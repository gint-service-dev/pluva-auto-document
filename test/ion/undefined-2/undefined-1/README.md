# 오토플래닝

## 오토플래닝

오토플래닝은 작업할 필지의 바운더리와 주행 조건을 기반으로, AB직진·A+직진·AB커브·자동 경로 생성 등 다양한 주행 모드의 작업 경로를 설계하는 기능입니다. 플루바 AI가 필지 형태와 작업 환경을 분석해 중복 작업과 빠짐 구간을 줄이고, 작업 시간과 운전 피로도를 동시에 줄일 수 있는 최적 경로를 제공합니다.
